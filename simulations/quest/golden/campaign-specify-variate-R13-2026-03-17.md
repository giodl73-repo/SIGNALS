# Variations — campaign-specify / Round 13

Five complete, runnable skill body variations targeting the v13 rubric.
Single-axis first (V-01, V-02, V-03), then combined (V-04, V-05).

**R12 diagnosis driving R13 axis choices:**

R12 V-05 is the first 37/37 ceiling under v13. It carries four structural novelties
above the R11 V-05 baseline:

- **C-42** — STEP 0C SOURCE BINDING: named block in the Voice Compliance Audit, before
  COLUMN-HEADER REQUIREMENT, explicitly naming Step 0c as the authoritative source for
  VCA cell values. Backward reference from audit → definition.

- **C-43** — STEP 0C CONTRACT ANCHOR: named block at Phase 3 entry, before any pitch
  version is written, requiring verbatim copy of the three Step 0c contracts. Contracts
  structurally present at point of use.

- **C-44** — CITATION FORM REQUIREMENT: named labeled requirement statement in Phase 2
  Defeating Do Nothing. The COLUMN-HEADER REQUIREMENT pattern applied to citations:
  template + named requirement statement = dual mechanism.

- **C-45** — ANCHOR GATE: named gate after STEP 0C CONTRACT ANCHOR at Phase 3 entry,
  explicitly validating the anchor is complete before pitch writing proceeds.

R12 V-05 establishes a four-position structural pattern for voice contracts:
(1) definition with FORWARD AUDIT NOTICE (Step 0c), (2) re-anchor before use
(Phase 3 STEP 0C CONTRACT ANCHOR), (3) gate at use point (ANCHOR GATE),
(4) backward source binding at audit (VCA STEP 0C SOURCE BINDING).

The same four-position pattern has NOT been applied to citations (stability record)
or to inertia costs. R13 probes whether parallel structural patterns for these other
artifacts yield v14 excellence patterns.

1. Does a descriptive-narrative phrasing register preserve 37/37? (V-01 — stability test)

5. Do V-02 + V-03 + a GATE CHAIN REQUIREMENT in Phase 0 — a named labeled requirement

## V-01 — Axis: Phrasing Register — Descriptive Narrative

**Hypothesis**: R12 V-05 uses a structured imperative register throughout: phase headers
give direct orders ("Do not advance"), gate labels prohibit ("Do not begin Phase 2 until"),
and the FORWARD AUDIT NOTICE commands ("Do not alter the contract phrasing"). V-01 tests
whether a descriptive-narrative register — phase headers describe what happens, gates are
framed as transition conditions rather than prohibitions, instructions describe requirements
rather than issuing commands — preserves 37/37. All four C-42–C-45 structural elements
are preserved with their exact labels. This is a ceiling stability test: if the dual-
mechanism and anchor-binding patterns survive register change, the v13 ceiling is
form-independent. Predicted: 37/37.
