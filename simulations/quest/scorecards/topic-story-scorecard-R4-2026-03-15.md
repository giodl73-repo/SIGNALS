## Excellence Signals

**V-01 / V-02 (shared)**

1. **Structural pre-composition outperforms instructional pre-composition** — V-01's Part 1/Part 2 split and V-02's named block sequence both make the analytic step a checkable deliverable. V-03's "write this before" is advice; the structural variants make it a schema violation to skip. This is the design move that locks C-15.

2. **Validity rules as failure tables** — V-02's block validity rules table (what each block fails on) creates a checklist the model can self-apply at composition time. V-01's hard stops serve the same function but as a prose list. Both are stronger than inline failure examples because they are inspectable outside the narrative flow.

3. **Posture statement as a required block, not a recommendation style** — Both V-01 and V-02 isolate the posture statement as a named, formatted unit inside the recommendation beat. This is what makes C-16 reliably pass: the posture isn't inferred from the recommendation tone — it's an explicit required output.

**Persistent gap across all three (C-11)**

None of the three variations require the explicit bridge sentence: "The pattern [stated in Block A] is why this recommendation verb was chosen." All three have the pattern appear in Beat 3 and the posture in Beat 5, but the logical derivation link — pattern → verb — is never required as a stated sentence. C-11 will remain PARTIAL until the prompt requires a traceability sentence in the recommendation beat that cites the pattern by name.
