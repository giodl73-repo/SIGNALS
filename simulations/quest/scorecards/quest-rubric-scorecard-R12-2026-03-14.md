### Excellence Signals — V-05

**Signal 1: Phase 0 dedicated closure gate is the structural guarantee for C-30**

V-05 is the only variation to achieve C-30 PASS. The mechanism: Phase 0 is a role-switched phase (Failure Analyst, not SetCoherenceAuditor) with its own three-gate exit condition (0-A: count >= 3; 0-B: skill-specific observable signal per mode; 0-C: reason for unusability per mode). The Phase 1 entry condition locks the failure modes list as "complete and closed — no new failure modes added during Phase 1." The Phase 2 derivation checkpoint row enforces the link per criterion: "a criterion that introduces a new failure mode not in Phase 0 fails this gate." This three-layer enforcement (Phase 0 exit → Phase 1 entry lock → Phase 2 per-criterion derivation check) is structurally tamper-proof — the failure modes list is enumerated before Phase 1 opens, closed when Phase 1 opens, and verified against at every Phase 2 criterion write. Role-obligation-embedded enumeration (V-01/V-02) provides only a PARTIAL because Phase 1 planning simultaneously enumerates and drafts without a closure gate between enumeration and derivation.

**Signal 2: Pre-role inertia framing (V-02) is a null result when structural gates already enforce failure-mode derivation**

V-02's enemy-first callout block before the role definition produces zero criterion improvement over V-01. When Phase 1 Gate 2 (imperative skill-specificity check), the per-criterion "Skill-specific criterion text" checkpoint row (STOP AND REWRITE), and the role-constitutive obligation ("Enumerate the most dangerous failure modes... before writing any criterion") already enforce failure-mode derivation structurally, placing the enemy-first block at prompt-parse time adds no measurable C-04 or C-11 enforcement. C-11 tests presence and position of the DR dominant-failure-mode sub-section in the output — not elaboration depth above the pre-role echo. V-02 = V-01 = 99.81. Inertia framing is redundant when the full structural stack already closes the generic-criterion gateway.
