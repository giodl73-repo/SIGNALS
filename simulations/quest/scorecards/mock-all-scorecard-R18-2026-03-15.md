### Discrimination Inversion Finding

V-01 (26/29) scores one point **above** V-02 (25/29), inverting the intended ladder at the bottom two rungs.

**Cause:** V-01 is single-stage, so C-35 trivially passes (C-35 = C-17; both PASS). V-02 is multi-stage with terminal-only inertia instruction, so C-35 FAIL. The C-35 cost in V-02 is not compensated by any gain elsewhere — both templates pass identical criteria across C-09 to C-33. V-01 thus fails 3 criteria (C-34, C-36, C-37) against V-02's 4 (C-34, C-35, C-36, C-37).

**Implication for rubric design:** The intended ordering V-01 < V-02 held in R17 because R17 V-01 and V-02 were both earlier-generation templates that failed C-22 through C-30 in different patterns. R18 templates carry the full structural feature set to the bottom rungs, removing the earlier-generation gap. If the round-bottom ladder position requires V-01 < V-02, a future round could fix the inversion by designing V-01 to also fail C-35 (multi-stage with terminal inertia, no preamble declaration, no labels) and V-02 to add a canonical declaration without labels.

### Excellence Signals — V-05 (29/29 Ceiling)

**ES-R18-01 — Gate citation as auditability shortcut:**
V-05 demonstrates that C-37 is a *readability property* of gate text, not a structural enforcement property. V-04 and V-05 enforce the same compliance requirements; V-05 makes compliance verification cheaper. A scorer confirming C-35 in V-04 must read the stage body instruction ("Per Declaration B: extend the inertia skeleton...") to understand what the gate enforces. In V-05, the gate itself says "per Declaration B" — the preamble declaration + gate citation is sufficient without traversing stage instruction bodies. The closed reference loop reduces the verification path by one traversal level.

**ES-R18-02 — Label assignment as the minimum step above unnamed declaration:**
The structural difference between V-03 (27/29) and V-04 (28/29) is entirely in two words added to the preamble: "Declaration A" and "Declaration B" as header labels. V-03 has the full declarative sentences for REAL-REQUIRED authority and per-stage inertia authority; V-04 adds only a named label to each. The label costs nothing in enforcement strength — the same rules apply. What it adds is addressability: stage gates can cite "per Declaration B" only because a label exists. C-36 is the label-assignment rung; without it, C-37 is structurally impossible.

**ES-R18-03 — Chain closure pattern:**
The full five-rung chain (C-32 → C-33 → C-34 → C-36 → C-37) moves compliance authority from presence at gate scope to closed-loop citation at document scope. Each rung reduces the traversal cost of verification by one scoping level:
1. C-32: gate names the block
2. C-33: gate uses the semantic identifier
3. C-34: preamble declares the identifier canonical
4. C-36: preamble labels the declaration
5. C-37: gate cites the declaration by label

V-05 closes all five rungs. A new chain reaches its ceiling when the reader's traversal path collapses to preamble + gate citation alone.
