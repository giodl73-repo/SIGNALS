### Excellence Signals

**What separates 185 (V-02/V-04/V-05) from 175 (V-01/V-03):**

The scoring gap is entirely driven by **Axis B** — the STRUCTURING-COST FRAME placed before the mechanism table. This contributes C-39 PASS (+5 pts) and C-37 PARTIAL (+2.5 pts). Axis A and Axis C contribute no scoring points against existing criteria.

**What Axis A adds (non-scoring, V-01/V-04/V-05):**

The IF/THEN consequence injection at three violation sites is a new enforcement pattern. Instead of "DO NOT interleave tiers," V-01/V-04/V-05 write: "IF any tier-N entry appears before all tier-(N-1) entries are written, the ROLE-TYPE-CLASSIFICATION block is invalid -- re-emit the full classification in strict tier order before proceeding." The consequence is named at the violation site, not deferred to the gate checkpoint. This is a first-class new pattern — the only prior consequence-chain in the spec is the FLAT-CASE-CLOSED self-repair (IF count < 2: write missing rows), which was a R15 signal. The IF/THEN at tier-interleaving, row-misplacement, and Quorum-omission sites generalizes this pattern to three additional violation points.

**What Axis C adds (non-scoring, V-03/V-05):**

The GATE-CONDITION-MET arrival block is a structurally novel pattern with no precedent in the rubric. It converts the gate from a one-sided declaration (departure site emits) into a two-sided contract (arrival site confirms receipt by quoting the exact departure line). The SOURCE-GATE field's exact-quote requirement is the mechanism: if the consuming section quotes the gate line verbatim, it forces the model to have the upstream gate line visible at the point of headcount production, seeding continuity. This pattern has the structure of a future criterion: "arrival-site artifact receipt acknowledgment at the consuming section of the highest-risk gate transition."

**Path to golden from V-05 ceiling:**

V-05 scores 185/295. Reaching 236 requires approximately 51 additional points. Highest-value gaps:
- C-17 + C-19 (interleaved rhythm-charter production with pair-count verification): +10 pts
- C-37 full PASS (add NET-COST-COMPARISON + SUB-SECTION-4-ANCHOR-SEQUENCE): +2.5 pts (PARTIAL → PASS = +2.5)
- C-33 + C-34 + C-35 + C-36 (GATE 3 charter audit chain): +20 pts
- C-41 (CHECKPOINT-INERTIA for STRUCTURING-COST FRAME positional enforcement): +5 pts
- C-16, C-27, C-28, C-30, C-31, C-32 (remaining structural enforcement criteria): +30 pts

The Axis C pattern (GATE-CONDITION-MET) is the strongest candidate for a new criterion in R19: it closes the arrival-site gap at GATE 2->3 and is the first instance of a two-sided gate contract in the spec.

The exact-quote constraint on SOURCE-GATE is the key mechanism: it forces the arrival section to have the upstream gate line explicitly in scope before any consuming content is written, seeding cross-gate continuity enforcement at the highest-risk transition (GATE 2->3, org diagram → headcount area derivation).
