## Gap Analysis: V-02 and V-04

**V-02 C-44 gap — `STAGE 2 COMPLETION GATE`**: The stage boundary label (`STAGE 2`) is navigational vocabulary that helps the author find the gate in the document; it is not an enforcement directive. C-44 requires the marker component to carry imperative vocabulary that tells the author what to DO before advancing. `STAGE 2 COMPLETION GATE` names the location of the gate, not the obligation at the gate. Compare: `PASS BEFORE ADVANCING` (V-01/V-05) or `CONFIRM` (V-04) — both are commands. `COMPLETION GATE` is a noun phrase.

**V-04 C-37 gap — bracket command in heading vs body**: The combined `[BRIDGE COMPLETION COMMAND]: CONFIRM -- HAVE BOTH...` heading achieves notable structural compression (A-25+A-27+A-33+C-44+C-45 in a single markdown heading) but at the cost of C-37, which requires the bracket directive to be in the gate body. The rubric explicitly anticipated this: "an output can pass C-39 while failing C-37 if the remediation command is embedded in the heading rather than in the block body." V-04's combined heading is the precise case that sentence described. The combined heading satisfies C-44 (imperative `CONFIRM`) but is structurally the wrong location for C-37.

## Excellence Signals — Top-Scoring Variations (V-01, V-03, V-05)

**Signal 1 — Three-part gate heading separates navigation, enforcement, and decision (V-01, V-05)**
Form: `[navigational label] -- [imperative enforcement] -- [binary question]`
Both V-01 and V-05 use `BRIDGE COMPLETION GATE -- PASS BEFORE ADVANCING -- HAVE BOTH...`. The first component (`BRIDGE COMPLETION GATE`) serves document navigation (author can locate the gate by heading scan); the second component (`PASS BEFORE ADVANCING`) is the enforcement directive C-44 requires; the third component is the binary decision C-39 requires. This three-part structure satisfies C-44 and C-39 simultaneously while retaining a navigational label V-02 tried to include but failed C-44 because it made the navigational label the ONLY pre-question component. Not yet formalized as a criterion.

**Signal 2 — C-46 axis vocabulary propagating into artifact section headings (V-05)**
V-05 names its bridge section headings `## Q3 (DO-NOTHING DEFAULT ACTOR BRIDGE)` and `## Q4 (DO-NOTHING DEFAULT TRIGGER BRIDGE)`. The C-46 compound axis vocabulary (`DO-NOTHING DEFAULT`) is not limited to bracket-labeled rule elements (A-16/A-19/A-31) — it propagates into the artifact-section headings themselves. C-46 currently requires the compound term to thread through bracket-labeled elements; V-05's pattern extends threading to non-bracket heading vocabulary, creating the deepest cross-criterion vocabulary coherence in R14. The gate heading artifact class `DO-NOTHING DEFAULT BRIDGES` is then naturally derived from these artifact names, creating a single vocabulary family from artifact section heading → artifact class noun in gate heading → rule label threading. Not yet formalized as a criterion.
