## Excellence Signals from V-05

**1. Pre-execution Phase 0 as a structural guarantee mechanism**
Running a mandatory setup phase (scout gate → inertia baseline → contract matrix) before any artifact begins converts three commonly-missed criteria (C-06, C-07, C-09) from probabilistic to near-deterministic. The model cannot "start writing" before completing the setup checklist.

**2. Per-signal routing table in the scout gate**
Instead of a binary "scout scan performed / not performed," V-05 produces a table mapping each found signal to the artifact that should incorporate it. This makes C-07 testable: if the signal appears in the routing table, it must appear in the named artifact. The routing table is evidence of intent; artifact incorporation is evidence of execution.

**3. Two-layer sequence integrity enforcement**
C-05 failures happen when the model writes Artifact 3 without re-reading Artifact 1. V-05 prevents this with (a) dependency checks at the start of each artifact and (b) a PM consistency check after all three are written. The second pass catches contradictions the first pass missed.

**4. Role handoff statements as artifact delimiters**
Explicit "PM hands off to Architect for DESIGN/CONSTRAINTS" statements make the spec's weakest criterion (C-02 technical depth) fail-safe. The handoff creates a named accountability boundary — the Architect cannot be silently skipped.

**5. Terminal consistency check as a completion gate**
The PM reviews all three artifacts for contradiction before the skill is "done." This produces a natural moment to surface cross-artifact contradictions (C-05) and missing inertia references (C-06) that were declared but then omitted.

## Observations

- **C-02 (spec technical depth) is the primary essential differentiator.** V-03 and V-04 both score well on recommended/aspirational criteria but fail golden because C-02 is PARTIAL. Any variation that treats spec section labeling as implicit rather than explicit risks this failure.
- **C-10 (anti-positioning) is underweighted in design but high-value.** Only 5 pts aspirational, yet V-01 and V-04 pick it up for free through audience framing and conversational register respectively. Easy win if explicitly prompted.
- **V-01 and V-02 tie at 82.5 but via different profiles.** V-01 is stronger on C-06 (inertia everywhere) and C-10 (audience framing); V-02 is stronger on C-05 (sequence integrity) and C-09 (contract matrix). They are not interchangeable — combining them produces V-05.
