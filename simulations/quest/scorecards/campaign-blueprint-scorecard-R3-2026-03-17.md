## Excellence Signals (V-02 and V-04)

**Signal 1 — Compound Phase 0:** Both V-02 and V-04 achieve the only 5 pts separating them from V-01/V-03 by embedding the artifact contract table *inside* Phase 0, alongside the Option 0 ledger and inertia baseline. When C-09, C-15, and C-16 obligations all live at the same mandatory gate, they become structurally co-dependent — an executor who completes Phase 0 satisfies all three at once, with no separate declaration step that can be skipped. V-01 and V-03 leave the contract implicit or per-artifact, creating the gap.

**Signal 2 — Inertia-first ordering (V-04 diagnostic answer):** V-04 puts the Option 0 ledger as item 1, before the feature identity in item 2. This doesn't change the score (V-02 also scores 130) but answers the R3 diagnostic: inertia-first ordering does not *add* score over neutral ordering when both have a ledger template. The ledger template is the load-bearing element for C-16, not the ordering.

**Signal 3 — Minimal sufficiency confirmed (V-05):** V-05's 4-field Phase 0 block (inertia baseline + Option 0 name + Option 0 cost + scout) achieves full C-15 and C-16 credit at 127.5/130. This answers R3 diagnostic question 3: Phase 0 word count does *not* correlate with C-15/C-16 coverage. The gate marker and propagation rule are the load-bearing elements. V-05 fails only C-09 (partial) because it distributes READ/WRITE/PRESERVE/CARRIES FORWARD across artifact sections instead of consolidating in Phase 0.

**Architectural conclusion:** The delta between 125/130 and 130/130 is exactly C-09. V-02 and V-04 earn it by placing the contract table in Phase 0. The simplest path to a canonical 130-pt form is V-05's minimal density with V-02/V-04's Phase 0 contract table added — a compact Phase 0 block with five fields (4 from V-05 + the contract table as step 5).
