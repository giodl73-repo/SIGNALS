### Excellence Signals — R6 Patterns Not Captured in C-01–C-24

Three structural patterns in R6 rise above the current ceiling. Each addresses the class of gap C-24 identified — "criterion specified but not enforced at the artifact boundary" — applied to three distinct artifact surfaces.

**Pattern 1 (Axis D): Evidence form verification gate — RULE-1 declarative taxonomy converted to mechanical write-time enforcement**
RULE-1 names valid and invalid evidence forms and assigns a violation consequence. Phase 7 Check 2b closes the remaining gap: each P1 no-change MANUSCRIPT OUTCOME must name at least one valid RULE-1 evidence form by category label before write. A rationale using only restatement or precedent fails Check 2b regardless of phrasing quality or length. This is the same class of gap that produced C-24 — "every prior check passes but editorial review fails" — now applied specifically to P1 evidence quality. C-03 declares the inertia rule; Check 2b enforces that the declaration reaches the artifact surface.

**Pattern 2 (Axis E): Cover letter Paragraph 1 R-ID traceability gate — vague change claims become a structural failure category**
Phase 6a instructs "update Paragraph 1 to reflect changes actually made" but cannot detect untraced claims. An author writes "We substantially revised the methodology" without naming R-03 or R-09 — the claim passes Phase 6a and Phase 7 Check 3. Phase 6a's traceability requirement (parenthetical R-ID references in each change claim) combined with Phase 7 Check 6 (list each claim and its R-ID reference — PASS/FAIL per claim) moves cover letter vagueness from an editorial observation to a structural gate failure. A change claim traceable to zero R-IDs names a revision not driven by any reviewer concern — detectable and blocking before write.

**Pattern 3 (Axis F): SHIFTED pre-commitment classification with type-specific re-examination — unstructured logging converted to binary with defined consequences**
C-23 requires HELD/SHIFTED logging. "SHIFTED: state whether the shift was correct and why" is an unstructured prompt that produces observations without closure. Axis F adds a binary: PRESSURE-DRIVEN (author conceded under rhetorical weight without new scientific evidence — original no-change position may still be valid) vs EVIDENCE-DRIVEN (author encountered specific new evidence — change is likely correct). Each type triggers a distinct re-examination: PRESSURE-DRIVEN checks whether RULE-1 evidence still supports no-change; EVIDENCE-DRIVEN checks whether Paragraph 2 accurately reflects the resolved disagreement. Phase 7 Check 4 Part B verifies all SHIFTED entries are classified and re-examined before write. The binary converts a SHIFTED event from a silent concession into an examined outcome — the same principle C-23 applied to the position itself, now applied to the interpretation of the shift.
