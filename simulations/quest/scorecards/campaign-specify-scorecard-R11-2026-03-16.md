**V12 candidate patterns** (from V-05, strongest source):

**Pattern A — FORWARD AUDIT NOTICE in Step 0c** (V-05 only): Step 0c carries a named forward reference to the COMPLETION INDEX Voice Compliance Audit immediately after the voice contracts are written. **Pattern B — Explicit COLUMN-HEADER REQUIREMENT in VCA** (V-05 only): The VCA instruction block before the table includes a named requirement: "The third column header must read exactly 'Step 0c Contract Verified'." Lists disqualifying alternatives ("Contract Check", "Voice Contract", "Verified"). Elevates C-39 from template-embedded pattern to structurally stated rule. This is distinct from C-39 itself (which only requires the header to be present) — a v12 criterion would require the explicit statement of the rule, not just the pattern.

**Pattern C — Explicit AUDIENCE-DISCRIMINATOR REQUIREMENT in VCA** (V-02, V-04): Named instruction block before the VCA table states the requirement that cell values must carry the audience identifier embedded in the contract phrase. Lists the disqualifying form ("Step 0c voice contract satisfied" applied generically). Parallel mechanism to C-38 template — instruction carries the rule explicitly where C-38 carries it structurally. Present in two variations, making it more stable than Pattern A or B.

**Assessment**: Pattern C is the most regression-stable (present in two variations). All three are additive — none conflicts with existing criteria or with each other. V-05 is the natural foundation for R12 because it contains Patterns A and B simultaneously.
