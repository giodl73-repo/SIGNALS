### Evidence Notes by Criterion

**C-05 (essential)** — The universal gap: all five variations either specify the `SURPRISING MOMENT` label in extraction or a `Moment Flag` column, but none embed the `SURPRISING (expected: X, got: Y)` format template co-located with the labeling instruction. V-01 has the Expectation Register (C-13 PASS) which creates the structural precondition for the link, but the extraction section says only "Flag one item per subject as SURPRISING MOMENT or CONFIRMING MOMENT and label it" — the format and register are in different sections. PARTIAL. V-02 has no expectation register at all, so SURPRISING/CONFIRMING labels cannot be tied to prior expectations by any structural mechanism. FAIL.

**C-13** — V-01 is the only variation with a required Expectation Register ("Expected to say / Expected to miss / Expected to confirm") declared per subject before any transcript. This is C-13 PASS by construction.

**C-22 (universal FAIL)** — No variation includes a Vocabulary Signature as a declared auditable contract (2–3 role-specific terms in a dedicated pre-interview field). V-04 mentions vocabulary difference as evidence ("The INCUMBENT and the FRICTION HOLDER will describe the same limitation differently — that vocabulary difference is evidence") but never requires a named term list per subject card. The aspiration is instructional, not structural.

**C-18 (universal FAIL)** — All variations provide `[Epistemic Standing]: GROUNDED / CONDITIONAL / INFERRED` — which is experience-proximity, not origin-of-claim. C-18 asks for `verbatim (quoted from transcript) / inferred (implied) / corroborated (consistent across subjects)`. These are orthogonal taxonomies; all five conflate them.

**C-23** — V-02 alone achieves PASS: the Subject Cards TABLE has an explicit "Evidential Function" row with column-per-subject, with instructions naming what the function declaration must contain ("structural job in the investigation… not their role title… not their disposition"). V-01, V-04, V-05 declare function at role-type level (slot descriptions) but not as per-subject card fields. PARTIAL. V-03 omits function entirely.

**C-24** — V-01, V-02, V-04, V-05 all earn PASS. V-04 has the most explicit language: "The INCUMBENT's objection about the new direction (which they may not have seen) is CONDITIONAL; the same objection from a subject who has piloted the alternative is GROUNDED. Name the difference." V-03 earns only PARTIAL — EPISTEMIC RE-WEIGHTING is instructed ("flag CONDITIONAL items and credit GROUNDED ones; show that blind-spot declarations changed how evidence is weighted") but the required form naming the standing type while weighting isn't mandated explicitly.

**C-21** — Universal PARTIAL. All variations track disposition arc and ask about the inertia barrier's fate, but none explicitly require the verdict "objection PERSISTED / TRANSFORMED / RESOLVED" tracing — they ask about friction being "fatal / solvable / neutralized" which is outcome-framing, not trajectory-tracing.

**C-19** — V-03 is the only FAIL. It prescribes a phase structure but never requires a one-sentence sequence rationale. V-01 requires "State the sequencing rationale in one sentence before the first subject card." V-02 has Section 0 SEQUENCE DECLARATION. V-04 and V-05 embed the rationale in their framing prose (incumbent first because "Their relationship to the status quo is the evidence baseline. Everyone else gets measured against it.").

**C-15** — V-03 earns PASS: six named phases (PHASE 0–5) each with explicit gate conditions whose checklists name specific criteria ("At least one subject carries Incumbent Designation = INCUMBENT"). V-01, V-02, V-04 earn PARTIAL: steps are numbered and functional but don't label their rubric ownership. V-05: FAIL — coaching prose implies ordering but no phase names.

**C-20** — V-03 earns PASS with five explicit gate checklists (GATE 0–5), each with binary [ ] conditions. V-02: PARTIAL — one gate declared ("Gate: Section 2 is complete when every cell is populated"). V-01, V-04, V-05: FAIL.

### Excellence Signals — What Made V-01 the Top Scorer

**What V-01 does that no other variation does:**

1. **Expectation Register as structural pre-commitment (C-13 PASS)** — V-01 requires a dedicated pre-interview register per subject ("Expected to say / Expected to miss / Expected to confirm") before transcripts begin. This is the only variation that creates the structural anchor for C-05's prior-expectation link. Every other variation requires a SURPRISING/CONFIRMING flag but leaves the prior expectation implicit.

2. **The gap V-01 still leaves (and what unlocks C-05 PASS):** The Expectation Register and the extraction `SURPRISING MOMENT` flag are in different sections (Step 0/1 vs Step 3) with no format template bridging them. C-05 requires the form `SURPRISING (expected: X, got: Y)`. If V-01 embedded that template in the extraction section — e.g., `SURPRISING (expected: [from Expectation Register], got: [what emerged])` — C-05 would PASS. The pattern: the format template must co-locate with the labeling instruction, not merely co-exist in the same prompt.

**What V-04 does that approaches C-24 most precisely:**
V-04 names the required form almost verbatim: "The INCUMBENT's objection about the new direction (which they may not have seen) is CONDITIONAL; the same objection from a subject who has piloted the alternative is GROUNDED. Name the difference." This is the closest any variation gets to C-24's required adjustment statement. Combining V-01's expectation register with V-04's C-24 language would produce a stronger candidate.
