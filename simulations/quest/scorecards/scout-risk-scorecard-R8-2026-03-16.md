**What made V-05 better than V-03 (+5 pts):**

- **C-17 PASS vs. PARTIAL** — typed From/To columns replace prose severity linkage, making violations cell-detectable
- **C-22 PASS vs. FAIL** — typed table columns vs. phase/prose structure
- **C-26 PASS vs. PARTIAL** — typed cells enforce rendering

**What made V-05 better than V-02 (+6 pts):**
- **C-21 PASS vs. PARTIAL** — named repair loops with count match
- **C-23 PASS vs. FAIL** — Repair Loop A, B, C labeled
- **C-25 PASS vs. PARTIAL** — Phase 2b is standalone dedicated phase

**Structural pattern driving the gap:** V-05 combines typed tables (V-02's mechanical enforcement) with named phase structure and repair loops (V-03's self-healing architecture). Neither axis alone reaches C-21+C-23+C-22+C-25 simultaneously. The combination is superadditive: phase labels make repair loop names trivially verifiable (C-23), and typed tables make format violations mechanically detectable (C-22, C-26).

**The one remaining gap (C-19 PARTIAL across all):** All variations choose structural enforcement over enumerated prohibition lists. Typed columns and imperative vocabulary ("MUST", "format violation") achieve zero-generic outcomes by making the type structure itself the gate. But C-19 requires three specific named forbidden phrases by name. V-05 does not enumerate them — it eliminates the need for them through typing. This creates a residual 1-point gap that a single added sentence could close.

## New Patterns Detected

Pattern 1 — **Tables-throughout schema separation**: When all three output sections (inertia, dimensional, interdependency) each use a typed table with a distinct schema, format violations become mechanically detectable register-wide without prose judgment. The existing rubric covers typed columns only for interdependency (C-22) and distinct anatomy for inertia (C-28) — a holistic criterion requiring all output tables to carry typed schemas is not yet codified.

Pattern 2 — **Imperative vocabulary as structural enforcement complement**: "MUST" / "format violation" framing transforms format instructions from guidance into enforcement declarations, making non-compliance explicitly named in the prompt's own terms. This is a prompt-language quality signal that correlates strongly with C-26 and C-11 compliance but is not currently a scoreable criterion in its own right.
