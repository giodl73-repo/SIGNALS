### Excellence Signals from V-05

1. **Attribute-absence template is load-bearing for C-17.** Positive instruction ("reference attributes") consistently allows assertion-form WHITESPACE findings. The Row C{N} -- {column}: {absent value} template plus NOT ACCEPTABLE assertion examples are both required — neither alone closes the gap.

2. **Named failure states must cover all gate types.** C-16 requires failure state naming for every gate — citation gate, anchor gate, and proof structure gate. Variations that named some but not all (V-01 missing proof structure failure; V-02/V-03 missing citation gate failure label) scored PARTIAL.

3. **C-14 requires the prohibition of the shortcut by name.** Prescribing "rerun both reductions" is PARTIAL unless the shortcut form ("update the finding") is explicitly declared insufficient by name. V-05 closes this with "or to 'update the CROSS-DIMENSIONAL conclusion' is a **proof rerun failure**" — naming both the shortcut and the failure state simultaneously.
