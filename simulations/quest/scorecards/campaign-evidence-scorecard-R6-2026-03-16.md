## Excellence Signals

**V-04 (INERTIA RULE + lifecycle gates)**: Introducing a fifth peer-tier governance rule — INERTIA — makes the verdict framework three-outcome (Ready / Status quo holds / Indeterminate) rather than binary. The "don't proceed" posture is now a *named outcome with its own rule*, not a default omission. Explicit entry/exit conditions per stage (word budget, entry condition, exit condition) prevent stage bleed structurally rather than relying on executor discipline.

**V-03 ([Stage N of 5] invocation trail)**: Appending `[Stage N of 5]` to each invocation tag creates a machine-readable audit trail. A reader can verify the coverage map claims by counting invocation tags — if SEQUENCING has five `[invoked at: S1..S5]` but only four `[Stage N of 5]` tags appear in the body, the gap is visible.

**V-05 (quantified gap in verdict)**: `{N} claims remain Low confidence` in the decision readiness sentence compresses both posture and evidentiary state into one sentence, making the verdict actionable rather than directional.

**V-01 (fully interrogative throughout)**: No passive tags anywhere — every invocation is a binary question the executor must answer before proceeding. This is the limit case of C-21.

## New Patterns for Round 7

**Pattern A — Stage-indexed invocation trail (V-03)**: `[Stage N of 5]` suffix on rule invocations makes the invocation sequence auditable without cross-referencing the coverage map. A missing suffix is a visible gap.

**Pattern B — INERTIA RULE as fifth peer-tier governance rule (V-04)**: Declaring "do not proceed" as the explicit default posture, with the status quo named as a legitimate competitor, produces three-outcome verdicts (proceed / status quo holds / indeterminate). This surfaces a class of briefs that would otherwise issue a silent "proceed by default" when evidence is weak.

**Pattern C — Entry/exit conditions per stage (V-04)**: Explicit entry condition (what must be true before this stage begins) and exit condition (what must be true before this stage ends) prevent stage bleed structurally. C-16 encodes zero-gap invocation; entry/exit conditions encode stage boundary integrity.

**Pattern D — Quantified gap in verdict sentence (V-05)**: Naming `{N} claims remain Low confidence` in the single-sentence verdict compresses evidence state into the posture statement. C-12 encodes compression to one sentence; this pattern adds quantification of the gap to the verdict itself.
