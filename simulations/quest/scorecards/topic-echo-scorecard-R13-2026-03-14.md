## Excellence Signals

### Signal 1 — Inertia preamble creates a document-level enforcement tier orthogonal to all structural criteria (V-05)

V-05's inertia preamble is a document-level description of what happens when the echo skill is not used: the findings summary. It names the default, explains why the default cannot produce an echo, and identifies the institutional cost (lost surprises → next team re-discovers them). This framing reinforces C-01 (surprise focus) before any schema or gate enforcement — the author understands the document's purpose, not just its schema. Critically, the preamble does not interact with any structural criterion: it is pre-manifest, register-neutral (declarative in any phrasing), and does not modify gate execution or pair inspection structure. V-05 confirms that document-level motivational framing is a fully orthogonal layer that can be added above the structural architecture without affecting any criterion.

### Signal 2 — Format uniformity across all three template layers (V-03)

V-03's numbered-list format uses sub-item (1)(2)(3)(4) identically for canonical examples, the ★BASELINE entry, and fill-in pair slots. The author never encounters a different template shape when moving from an illustrated example to a blank pair to the pre-populated baseline — all three contexts use the same syntactic container. This is a stronger uniformity guarantee than step-labeled field blocks (which vary slightly between the illustration layer and the fill-in layer in prior formats). The format uniformity removes a class of potential confusion about which fields apply where.

### Signal 3 — C-30 universalization confirmed: the symmetric architecture is design-stable (all five)

The symmetric three-layer architecture (live inspection / illustration / baseline, each with analytical + synthetic components) holds across conversational register, lifecycle emphasis, numbered-list format, and three-axis composition. No surface-layer variation is sufficient to break the C-30 property. R13's uniform C-30 PASS confirms that template-complete baseline entries are not a V-05(R12) anomaly — they are a structural expectation that survives all known variation families.
