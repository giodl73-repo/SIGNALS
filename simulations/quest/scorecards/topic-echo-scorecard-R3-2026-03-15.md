## Excellence Signals from V-03

**Pattern 1 — Role-based counterfactual induction**
The Archaeologist role induces C-09-style thinking without requiring an explicit "without this signal, we would have..." clause. By asking "what would the next team get wrong?" the role positions every entry as a past-decision-avoided, even when the entry doesn't use counterfactual syntax. The mechanism is: role assignment → frame shift → counterfactual outputs. Future variations targeting C-09 should test whether the explicit clause is necessary if the role already induces the framing.

**Pattern 2 — Survivability filter replaces vividness filter**
The Archaeologist's test ("would this finding correct a false assumption still carried in the current design materials?") is structurally distinct from the prediction test ("would this have been predictable?"). It filters for ongoing relevance rather than original surprise magnitude. This means an entry that was genuinely surprising at discovery but is now obvious from the current spec would be excluded — prediction-test-based variations would include it. The survivability filter produces a more durable artifact.

**Pattern 3 — Discovery-to-correction vocabulary flip**
Renaming fields from discovery vocabulary ("Prior assumption," "Finding," "Design impact") to correction vocabulary ("The assumption still carried," "The correction," "Why it still matters") enforces C-06 (future-team framing) at the structural level. A model filling out "The assumption still carried" is forced to think about a new team's priors; a model filling out "Prior assumption" can reason from the original team's perspective. Vocabulary determines perspective more reliably than repeated framing instructions.
