## Excellence Signals from V-01

**table-enforces-column-completeness**: A named table column with enumerated values (CORRECT/WRONG/PARTIAL) is structurally impossible to skip. Prose fields can be omitted silently; a column cannot. This is why V-01 earns C-14 while V-02 earns only partial.

**ratio-template-propagation**: Modeling the exact output format inline (`N correct / M total = X%`) eliminates count-only responses. Models fill in the template as given.

**namespace-table-over-namespace-prose**: A table row per namespace forces completion of all 9; prose headers per namespace allow the model to truncate without visual gap. V-01 earns C-16 full; V-02 earns partial.

**exact-recommendation-template**: Providing the template verbatim with fill-in slots (`Addresses [Gap/Echo] by [practice change]`) is the only reliable way to get C-17. Instructing a model to "name what it addresses" produces prose that approximates the structure but misses the bracket-slot enforcement.
