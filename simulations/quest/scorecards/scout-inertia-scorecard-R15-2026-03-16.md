### Excellence Signals

**From the top-scoring variation (V-02, reference form):**

1. **Single-axis isolation as R15 design principle**: V-02 adds only A-35 (blockquote) to a scaffold that already carries A-34 and A-36. The reference form for any new criterion should be the minimal change on a confirmed scaffold — one new element, all others carried. This keeps the causal attribution clean and the next round's hypothesis specific.

2. **Blockquote-form universality confirmed across four scaffolds**: V-02, V-03, V-04, V-05 all apply blockquote form in different A-23 register contexts (bracket-suffix, domain-prefix, bracket-prefix, all-caps). None fail. The `>` prefix is a markdown structural operator that is orthogonal to the phrasing register of the rule labels — it does not require alignment with the surrounding register vocabulary.

3. **A-36 ordinal-agnosticism**: V-01 and V-03 confirm that `SECTION N` satisfies A-36 with equal force to `STAGE N`. A-36's pass condition is the presence of an ordinal position index in the heading name, not the specific word used. This means A-36 applies cleanly to any template structure that uses ordinal numbering for its organizational units.

4. **`FAILURE POINTS` and `FAILURE SURFACE` extend the A-34 vocabulary**: Beyond R14's confirmed `STRUCTURAL WEAKNESSES` and `WHERE X BREAKS`, R15 adds two explicit failure nouns (`FAILURE POINTS`, `FAILURE SURFACE`) as confirmed A-34 forms. The criterion is satisfied by any phrase containing an explicit failure-context word in the FM Inventory heading subtitle — the vocabulary is open-ended, not locked to specific forms.

5. **V-04's minimal-prefix A-36 form is a valid but weaker pass**: `STAGE 2 --` as a bare ordinal prefix before a combined command heading satisfies A-36 (position index present) but produces a heading name of just `STAGE 2` with no gate label. Future templates should prefer `STAGE N COMPLETION GATE` or `SECTION N COMPLETION GATE` for full semantic clarity. The minimal form passes the criterion but sacrifices self-contextualization.
