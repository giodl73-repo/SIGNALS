### Excellence Signals

From the tied top (V-04 as the most comprehensive):

- **Layered structural constraint pattern:** Setup declares obligations → per-artifact reminders enforce at execution point → gated reflection phase enforces post-production timing. Three-layer defense is harder to skip than any single instruction.
- **Conviction type as matrix row:** Naming the conviction type for each artifact in the setup matrix makes the distinction between factual, optionality, and emotional work visible before execution begins — reducing C-14 duplication risk even if C-14 itself doesn't yet score FULL.
- **FINDINGS as named post-phase:** The CAMPAIGN REFLECTION naming and gate ("not before production begins, not during production") transforms a clause-based instruction into a phase separator the executor can see in output structure — a structural honesty mechanism.

None of these patterns become new criteria because C-16/C-17/C-18 already describe the gap they point toward: the setup phase needs a **named-entity slot** for the do-nothing option, not just an inertia description.
