## R11 Design Notes

V-01 is the R11 full-stack baseline. Integrates all R10 structures (Phase 3 STOP CONDITION,
three-mode *Prevents:* at all phases, role-constitutive obligations preamble, five-column
planning table with FROM/TO secondary-effect column header, V-ID citation gate at Phase 1,
uniform imperative stops, sibling positive text anchor in primary-effect field, per-variation
checkpoint with predicate-based set-level rows) plus one new structural layer:

**Phase 2 Variation Completion Manifest (C-35 candidate):** After all five per-variation
checkpoints pass and before the Phase 3 STOP CONDITION is evaluated, the model must fill a
structured manifest table (V-ID rows, "Checkpoint status", "All checks passed?" column). A
blank row in the manifest blocks Phase 3 entry as a structural gate -- not merely advisory.
The manifest creates a durable evidence record of Phase 2 completion, distinct from the Phase
3 STOP CONDITION (C-29), which asserts completion without producing a checkable artifact.

Motivation: R10/V-01's Phase 3 STOP CONDITION requires "all 5 variation bodies confirmed
complete in Phase 2" but provides no structural record for the model to populate and verify.
The manifest closes this gap: it is the STOP CONDITION's evidence base, not a restatement of
it. C-35 candidate would require the manifest to be present as a structural table, populated
before Phase 3 begins. V-02 ablates the manifest (retaining the STOP CONDITION) to test
whether the table adds enforcement value beyond the STOP CONDITION's declarative gate.

Evidence ladder:
- V-02 (C-35 ablation, no manifest): predicts manifest absence at Phase 2/3 boundary
- V-03 (C-31 ablation, directional-only predicted-site column): predicts weaker falsification specificity
- V-04 (C-30 ablation, absence framing in primary-effect): predicts lower cross-variation anchoring
- V-05 (V-02 x V-03 combination): superadditivity test for C-35 + C-31 joint removal
