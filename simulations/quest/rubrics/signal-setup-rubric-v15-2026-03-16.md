**What changed in v15:**

Three new aspirational criteria extracted from Round 16 scorecard patterns:

| ID | Pattern | Source |
|----|---------|--------|
| C-36 | Preamble uses dominance-verb construction for inertia ("wins," "collects") | V-02 PASS C-13 — "Inertia wins the default choice" vs V-01 "rule governs" |
| C-37 | Preamble explicitly names team configuration as **choosing a side** | V-02 PASS C-16 — "teams choose a side" in text, not just implied by context |
| C-38 | At tool-specific decline gates, winning party is a behavioral force, not a tool name | V-01 FAIL C-33 at GATE 6B — "Copilot wins" fails; "inertia wins via Copilot's default" passes |

**Denominator: A=35 → A=38**

The C-36/C-37 pair has a clean progression: C-36 establishes the dominance verb register (inertia actively wins), C-37 establishes the team as active agent (teams explicitly choose). C-38 is a sub-case of C-33 with a sharper scope rule — any gate failing C-38 also fails C-33, and scorers apply C-38 only at explicitly tool-scoped gates.
