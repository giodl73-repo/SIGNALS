**v17 changes from v16:**

**Two new criteria extracted from R18 excellence signals:**

**C-45 -- Session-Open Phase Roster** *(r17 aspirational, 2.5 pts)*

Session-open blockquote names all 5 phases in sequence (Pre-Phase → Phase 0 → Phase 1 → Phase 2 → Phase 3), giving the model a forward map before execution begins. Any ordered notation form passes (arrow sequence: V-01/V-04; numbered list with output labels: V-05). V-02 and V-03 have a valid session-open tag but no roster -- confirmed as the contrast baseline.

**C-46 -- Session-Close Completeness Ledger** *(r17 aspirational, 2.5 pts)*

Session-close blockquote names each of the 5 phases individually rather than a generic count ("all five phases executed"). Two passing forms: named-ledger (each phase listed with brief output label, V-02/V-04) and echo-completion (each phase listed with its specific close-output echoed, V-05 -- richer variant). V-01 and V-03 use the generic count -- confirmed baseline contrast.

**Independence confirmed:** C-45 and C-46 are coordinate, not floor/ceiling. V-01 passes C-45 only; V-02 passes C-46 only; V-04 and V-05 pass both.

**Lineage position:** C-45 and C-46 are a new session-level branch, operating above all phase boundaries. They are not descendants of any single existing criterion.
