**Two new aspirational criteria from R12 excellence signals:**

**C-44 — Per-signal annotated inventory with role classification** (extends C-39, C-42)
V-04 showed that the ceiling can be *derived* rather than *estimated*: a mandatory Phase 1 annotates every signal with both evidence phase AND a role taxonomy (Primary/Supporting/Contextual). The C-42 declaration is extended to include Primary signal counts by phase, making the ceiling auditable by inspection. PASS requires per-signal annotation + role taxonomy + declaration extension. FAIL if phase annotation only, or if inventory is a summary rather than per-signal labels.

**C-45 — Two-dimensional ceiling table (phase × orthogonal coverage dimension)** (extends C-39)
V-05 showed that a second orthogonal dimension (inertia coverage: whether any signal compares against the status quo) can independently shift the ceiling at a fixed phase level. At Test phase: inertia-absent = 45, inertia-present = 60. The C-42 declaration is extended to name both dimension values. FAIL if ceiling varies by phase only, or if the second dimension lacks distinct intersection ceiling values.
