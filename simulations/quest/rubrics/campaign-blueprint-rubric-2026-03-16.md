**Summary of criteria:**

**Essential (5)** — all must pass:
- C-01: All three artifacts produced (spec + proposal + pitch present)
- C-02: Each written to its canonical path
- C-03: Consistent topic identity across all three
- C-04: Execution order respected (spec → proposal → pitch, no forward refs)
- C-05: Each sub-artifact meets its own minimum structure

**Recommended (3):**
- C-06: Proposal reflects spec decisions, no re-invention
- C-07: Pitch crystallizes the proposal's recommended option
- C-08: Campaign opens with setup summary, closes with package summary

**Aspirational (2):**
- C-09: Narrative arc — each artifact amplifies conviction without repeating content
- C-10: Scout signal pull is visible in the artifact that benefits from it

Golden threshold: all 5 essential pass + composite >= 80. The failure patterns section calls out the most likely failure modes: partial campaign, topic drift, out-of-order contamination, thin sub-artifact.
