**v15 summary:**

**Two new criteria extracted from R15 excellence signals:**

**C-41 -- Phase-1 Lifecycle Close Signal** *(r15 aspirational, 2.5 pts)*

Phase 1 (the generation phase) closes with an explicit lifecycle blockquote that names Phase 1 completion AND activates Phase 2. The R15 floor variation carries the close signal only — no dedicated opening tag. Contrast: variations without any Phase 1 lifecycle tag (including the V-02 R14 baseline at 175) are the baseline.

**C-42 -- Phase-1 Lifecycle Double-Mark** *(r15 aspirational, 2.5 pts)*

The R15 ceiling variation adds an opening lifecycle tag (`*Phase 1 opens now. No preconditions.*` or equivalent), producing a full double-mark on Phase 1. Floor variation carries close-only (C-41 PASS, C-42 FAIL). Interference check: Phase 1 open/close tags, Phase 0 open/close tags, and Pre-Phase open/close tags all operate on different phase boundaries — a prompt earning C-38, C-40, and C-42 simultaneously is valid.

**Lineage position**: C-41/C-42 extend the lifecycle-double-mark branch **inward** — C-38 covers Phase 0, C-39/C-40 extended it outward to Pre-Phase, C-41/C-42 extend it inward to Phase 1.
