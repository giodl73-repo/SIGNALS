**Two new aspirational criteria promoted from R19 V-05 excellence patterns:**

| ID | Pattern | Condition |
|----|---------|-----------|
| C-41 | `phase-domain-pair-sub-question-notation` | Stage 2 sub-questions carry explicit `(Domain / Phase)` pair labels in each sub-question's own label — e.g., `"1a (Commerce / PRE-DEPLOY)"`. Domain-only labels under a phase section header (`"1a (Commerce):"`) do not pass. The pair label forces phase assignment and makes a missing phase visually obvious; V-01's post-validate gap in R19 traced directly to domain-only notation. Requires C-40. |
| C-42 | `two-stage-only-path-ceiling` | Path-diagnostic criterion: C-37–C-40 all pass AND C-14 + C-21 both fail. Confirms the two-stage-only ceiling (~160/260); the +20 from C-37–C-40 stacks only on a prose structural-adoption base where C-21 passes. Deactivated on the R20 target path (prose structural-adoption + two-stage), where C-21 and C-14 pass and C-42 correctly fails. Analogous in role to C-34 and C-36. |
