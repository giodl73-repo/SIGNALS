## What changed v3 -> v4

6 new aspirational criteria (C-19 through C-24), each traceable to a specific R3 evidence signal:

| ID | Criterion | Source signal | Distinction from prior criteria |
|----|-----------|---------------|---------------------------------|
| C-19 | **Structured four-part hypothesis schema** | V-01 hypothesis: "criterion ID, direction, mechanism, and failure condition" | C-11 requires one failure condition; C-14 requires one criterion ID. C-19 requires all four parts in a single compositionally structured entry. |
| C-20 | **Dual mechanistically-distinct failure conditions** | V-02: "two modes -- no C-01 improvement vs R2, or gate implemented as post-generation review (independently fails C-15)" | C-11 requires one failure condition. C-20 requires two: one for outcome failure, one for implementation failure naming a separate criterion. |
| C-21 | **Rubric-ID exception citation for combination passes** | V-04 + V-05: "C-04 exception explicitly invoked" | C-04 already allows combination exceptions. C-21 requires the excepted criterion be cited by ID, making the waiver a traceable protocol act rather than an implied permission. |
| C-22 | **Criterion-keyed combination roadmap rows** | C-09 roadmap: "criteria C-01, C-04, C-07" and "criteria C-03, C-07, C-14" per row | C-09 requires rationale per row. C-22 requires specific criterion IDs per row, converting the roadmap from narrative to experimentally actionable. |
| C-23 | **Phased prompt architecture** | V-04: PHASE 1 (commitment) -> PHASE 2 (generation + gate) -> PHASE 3 (findings), each with declared responsibility | C-15 requires an inline gate. C-23 requires the gate to live inside a labeled, numbered phase container with transition points named. |
| C-24 | **Pre-generation variation score predictions** | V-05 PHASE 1: "champion inventory + variation map with score predictions" | No prior criterion requires pre-generation quantitative prediction. A champion inventory alone (C-16) doesn't pass -- each planned variation needs a per-criterion directional prediction. |

**Scoring formula**: aspirational denominator bumped from `/10` to `/16` (10 prior + 6 new). All bands and weights unchanged.
