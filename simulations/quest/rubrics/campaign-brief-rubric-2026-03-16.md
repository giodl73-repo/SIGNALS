**10 criteria across 3 tiers:**

**Essential (C-01–C-05)** — the minimum for a useful output:
- C-01: All four sub-skills invoked in order
- C-02: Existing signals enumerated by namespace (not just "signals exist")
- C-03: Missing signals explicitly named (not silence on gaps)
- C-04: Narrative arc synthesizes what signals say *together* — not a flat list
- C-05: Topic registered with name, date, intent

**Recommended (C-06–C-08)** — what separates good from adequate:
- C-06: Explicit readiness verdict (ready / not ready / conditional)
- C-07: Gap list prioritized — blocking vs. optional
- C-08: Scannable dashboard format with visual hierarchy

**Aspirational (C-09–C-10)** — raise the bar once stable:
- C-09: Session delta — what changed this session
- C-10: Narrative confidence level with rationale

The core diagnostic pair is C-02/C-03 — a dashboard that can't tell you what you *have* and what you're *missing* isn't a dashboard. C-04 is the differentiator: if narrative synthesis is absent, the orchestration over plain `topic-status` has no value.
