- **C-44 vs C-42 / C-21 / C-34**: C-42 tests whether a named binding block exists before Q1, creates ALIGNMENT STATE as a structural record, and mandates that the three artifacts — COST LEDGER row (a), Phase A Q1 analysis domain, B2 chain substrate — name the same infrastructure condition class.
