## What changed in v13

**C-29** (new): Inline FM-NN acknowledgment log as a named table artifact — extends C-28. The excellence signal from Round 15 V-02 is that prose bullet acknowledgments within stage sections satisfy C-28, but organizing those acknowledgments as a pre-declared append-log table (T-07 pattern) where each stage block appends rows is structurally distinct: cumulative evidence becomes visible as a referenceable table artifact before the post-trace resolution section commits.

**C-30** (new): SLA domination point as a named dedicated table artifact — extends C-22. The excellence signal from Round 15 V-02 is that a domination point statement appended as prose to the boundary inventory table satisfies C-22, but presenting it as a separate named pre-declared table (`| Domination Boundary | Exact Cumulative SLA% | Justification |`) converts the finding into a cross-referenceable artifact under C-24's navigational contract.

**C-31** (new): Structured recovery framing template with actor and cadence fields — extends C-16. The excellence signal from Round 15 V-04 is that step-level citation (C-16) identifies *what* the fallback is but not *who* performs it or *when*. The `Revert to [IB-NN Step] performed by [Role] at [frequency/trigger]` template enforces all three fields in a single Recovery Framing column, making incomplete fallback prescriptions structurally visible as missing-field gaps.
