# Quest Rubric — campaign-evidence (v20)

## Essential Criteria (60 pts)

| ID | Criterion | Category | Pass Condition |
|----|-----------|----------|----------------|
| C-01 | **Multi-stage campaign (5 stages)** | structure | The output was produced by a campaign of at least 5 named evidence-gathering stages executed in declared order. A single-pass synthesis does not pass. |
| C-02 | **Evidence before hypothesis** | structure | Web search and intelligence stages complete before hypotheses are declared. A campaign that states hypotheses in the prompt preamble or before any evidence stage executes does not pass. |
| C-03 | **Hypotheses with falsification conditions** | correctness | Each hypothesis carries an explicit falsification condition — a stated criterion that would cause the hypothesis to be rejected. Hypotheses without falsification conditions do not pass. |
| C-04 | **Output is self-contained** | format | The output is a single coherent document (not a collection of raw dumps). It has a title, topic context, and synthesized narrative — a reader unfamiliar with the run can understand what was investigated and what was found. |

| ID | Criterion | Category | Pass Condition |
|----|-----------|----------|----------------|
