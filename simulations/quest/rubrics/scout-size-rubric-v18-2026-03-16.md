**C-44 — Self-test affirmative branch prescribes targeted corrective scope by name**

This upgrades the self-test from a two-stage diagnostic (failure class + contract label) to a three-stage repair guide (failure class + contract label + structurally grounded corrective scope) — from detector to prescriber.
