**Rubric summary — corps-leaderboard v1**

**Essential (5)** — all must pass for golden:
- **C-01** Coverage — complete topic scan, no silent omissions
- **C-02** Correctness — all 5 achievement types evaluated per topic (First Signal, Signal Depth, Full Sweep, Solo Pioneer, Team Topic) with exact names
- **C-03** Correctness — all 3 team milestones present with verbatim names ("first team signal", "shared coverage", "debate starter") + status + evidence
- **C-04** Coverage — ranked contributor leaderboard always present (explicit "no metadata" row if attribution unavailable)
- **C-05** Behavior — 3+ next actions, each naming a specific namespace+topic AND the exact achievement/milestone it unlocks

**Recommended (3)**:
- **C-06** Format — earned vs locked achievements structurally separated
- **C-07** Depth — 1-away gap callouts with exact numeric gaps
- **C-08** Behavior — graceful empty-workspace handling (all sections still produced)

**Aspirational (2)**:
- **C-09** Depth — one cross-topic/cross-contributor synthesizing insight with specific names+numbers+implication
- **C-10** Depth — stagnation pattern named and at least one action explicitly tied to breaking it

**Golden floor**: all 5 essential + 2/3 recommended = score of 80.
