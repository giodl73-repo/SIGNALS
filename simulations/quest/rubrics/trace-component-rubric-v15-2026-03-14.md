**Three new criteria promoted from R14:**

| # | Name | Source | Mechanism |
|---|------|--------|-----------|
| **C-46** | Epistemic-Register Question Framing | V-05 null-hypothesis framing ("what is the epistemic status of a traversal hop before any evidence is applied?") | Requires causal questions to be phrased at the epistemic level so the structural answer is not implied by the question vocabulary — a model answering an epistemic question cannot satisfy it by substituting question labels into schema fields |
| **C-47** | Causal-Completion Gate Before Schema Authorship | V-05 "answers required before placeholder design" sequencing | Requires the prompt to enforce causal-phase completion as a structural precondition of schema-phase entry — a phase boundary, not an advisory instruction; prevents post-hoc rationalization where the model designs-then-explains |
| **C-48** | Null-Hypothesis Register in Causal Question Set | V-05 "null-hypothesis question set + three-property question set in sequence" | Requires at least one null-register question addressing the epistemic status of the default/null case, appearing as a sequentially prior block before departure-property questions — the null baseline must be established before departure conditions can be evaluated as departures from it |

**Ceiling: 164 → 170 pts** (3 new × 2 pts)

**Orthogonality:** All three are independent. C-41/C-42 satisfy the existence conditions for causal questions and blank authorship but specify nothing about question register, phase ordering, or null/departure decomposition. C-46 can be satisfied without C-47 (epistemic questions, interleaved phases). C-47 can be satisfied without C-46 (sequenced phases, structural-register questions). C-48 can be satisfied without C-46 or C-47 (null register present, structural framing, unsequenced). The cluster formed by all three — C-46 ensures derivation over copy, C-47 ensures explanation precedes design, C-48 ensures the null baseline is grounded before departure conditions are asserted — is the R14 high-water form. V-05 is the sole variation carrying evidence of all three.
