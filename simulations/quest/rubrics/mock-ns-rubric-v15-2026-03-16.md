> "There is no step that precedes this step's TOPICS.md status line emission."

Eleventh declarative failure mode. Expletive `There` is subject; the step is genitive possessor inside "this step's TOPICS.md status line emission" — never in nominative-subject position. C-38 does not fire (different expletive + no cleft-focus NP). C-39 does not fire (no wh-nominalization subject).

### C-41 — C-35 by-PP scope boundary (R14 V-01)
> "The TOPICS.md status line is emitted by this step before any other step begins."

C-35 (relative-clause-agent displacement) requires the step to appear as agent inside a subordinate clause with a verb ("that this step emits"). A by-PP ("by this step") is a phrase-level adjunct — no verb, not a clause. Passive + by-PP fires C-33 only; C-35 passes. Closes the evaluator trap where hypothesized C-35 on V-01 was wrong.

**Failure chain: C-33 only → 144/146**
