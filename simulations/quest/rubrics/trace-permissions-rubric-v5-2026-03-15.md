Two new criteria extracted from Round 4:

**C-18 — NOT POSSIBLE verdict closure mechanism citation** (from V-02 C-17 primary axis)
C-17 requires NOT POSSIBLE verdicts exist. C-18 requires each one names the structural fact that closes it — the configuration state confirmed absent, the boundary enforced, or the mechanism confirmed non-applicable. "NOT POSSIBLE" without a cited structural basis is an assertion indistinguishable from an unexamined vector that received a negative label.

**C-19 — Per-role escalation roll-up assertion** (from V-02 C-11 evidence)
C-11 requires individual vector rule-out statements. C-19 adds a per-role aggregate closure: a statement naming the count of vectors assessed and a composite finding. Without it, a reader must count and aggregate across vector rows — the synthesis the output should provide is left to the reader.
