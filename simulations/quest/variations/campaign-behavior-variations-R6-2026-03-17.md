## Axes explored in R6

**Single-axis (3):**

| Variation | Axis | Target gap |
|-----------|------|------------|
| **V-01** | Q1–Qn interrogative calibration | C-18 — the only criterion that blocked all 5 R5 variations from a perfect score |
| **V-02** | SYSTEMIC phase enumeration | C-21 — new in v5; mandates corroborating phase names in every SYSTEMIC flag |
| **V-03** | State-anchor role sequence | Unexplored execution order; trace-state first pre-classifies blast radius from shared-state topology before any access or contract analysis |

**Combinations (2):**

| Variation | Axes | Design intent |
|-----------|------|---------------|
| **V-04** | Q1–Qn + permissions-anchor | R5 V-01 base (89/90) with C-18 gap closed; adds C-21 phase enumeration and explicit tiebreaker (C-20) in the same pass |
| **V-05** | Full 13-criterion attempt | Permissions-anchor + DEPTH asymmetry + CONFIRMED/RUNTIME ANOMALY throughout + Q1–Q5 interrogative calibration + SYSTEMIC phase enumeration + atomic 7-field blocks + tiebreaker; every new v5 criterion has two reinforcement points |

**Key structural changes from R5:**

- **C-18 addressed:** V-01/V-04/V-05 replace step-block calibration with Q1–Q5 (each question has a named re-grounding target and an explicit answer)
- **C-21 addressed:** V-02/V-04/V-05 enforce `SYSTEMIC: yes -- phases: [name], [name], [name]` in definitions, per-phase tables, calibration, and consolidated blocks — binary flags rejected by name
- **C-19/C-20 reinforced:** atomic 7-field finding blocks and the three-key tiebreaker protocol are stated in definitions, not just in CONSOLIDATE
- **V-03 is the only variation without Q1–Qn** — it tests whether the state-anchor execution order independently changes blast-radius topology

**Expected ceiling:** V-04 and V-05 are designed to pass all 13 aspirational criteria (first 90/90 candidates). V-01 is the minimum-change path to close the R5 universal gap.
