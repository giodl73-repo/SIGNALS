## Key C-44 Structural Analysis

**Single-phase anchor** (V-01, V-02): The gap field's own production contract defines valid content as "a dimension the Step 5 basis did not verify" — this is structurally equivalent to "Phase 1 did not confirm" but anchored to Step 5 as the single-phase confirmed-set analog.

**Two-phase anchor** (V-03, V-04, V-05): "Address a dimension Phase 1 did not confirm" is grounded in Phase 2's positive charter definition — derivable from the contract itself, not from the non-overlap rule alone.

**C-45 candidate** (V-04, V-05): The REMEDIATION field's examples are elevated from parenthetical illustrations to a named type schema. V-04 requires the model to name which category it targets; V-05 requires the gap itself to carry the category label — making the correction's dimension class a structural property of the gap output rather than an inferred quality.
