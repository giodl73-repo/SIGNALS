The key R3 structural insight: C-14 and C-15 are both **argumentative completeness mechanisms** — they require the model to compute loss before writing, then cite that loss explicitly in the recommendation. Variations that define the loss in Phase 0 *as a field the recommendation must reference* (V-01, V-04, V-05) are more reliable than variations that expect the recommendation to recall it from context (V-03).
