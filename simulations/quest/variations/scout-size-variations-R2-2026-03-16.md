## What R2 targets and why

R1 produced two co-champions (V-04 interrogative, V-05 combination) at 100/100 on the old rubric. The v2 rubric adds C-11 and C-12. Both were likely already satisfied by R1's champions — but R1's V-01/V-02/V-03 had no structural guarantee for either. R2 closes that gap across all five variations.

### Single-axis bets

| Variation | Axis | Core mechanism |
|---|---|---|
| **V-01** | Confidence anatomy split | Two labeled mandatory fields — BASIS (what IS known) and GAP (what is NOT known) — make C-04 and C-11 structurally distinct. You cannot fill only one. |
| **V-02** | Single-condition template | `Tier rises to [LEVEL] if [single named condition].` forecloses hedges ("several factors") and wrong-dimension failures ("if the timeline slips") because neither can fill the template. |
| **V-03** | Inertia-as-anchor | Inertia is the structural opening with a mandatory 3-word cost direction verdict (`cheaper / comparable / more expensive`). Closes the "mentioned in passing" failure mode for C-03. |

### Combinations

| Variation | Axes combined | Key bet |
|---|---|---|
| **V-04** | V-01 split + V-02 template + R1 V-05 table base | TABLE 3 gets Basis/Gap as distinct rows; TABLE 2 tier sensitivity rows use the fill-in-the-blank template. Two new v2 criteria covered structurally inside the table schema. |
| **V-05** | All axes + anti-pattern block + 12-criterion self-check | Adds a table of **failing forms** (showing what "several factors" and "restates basis" look like) and a 12-item self-check covering every rubric criterion. The most complete defense against all named failure modes simultaneously. |

### Key structural differences from R1

- **C-11**: R1 used "What is unverified?" and "Open gap" row implicitly. R2 makes the Basis/Gap distinction *explicit* — naming "These are different questions" — and adds an anti-pattern showing what a conflated confidence section looks like.
- **C-12**: R1 used "What SINGLE condition..." phrasing. R2 uses a fill-in-the-blank template plus a table of failing forms that cannot be answered with hedges. The failing-forms table is the new mechanism not present in R1.
