### Key structural decisions across all five

**C-13 placement:** Each variation declares the vocabulary rule as a standalone section, not embedded in Stage 1 prose. V-01 calls it `Closed-Vocabulary Rule`, V-02 `Dimension Rule`, V-03 `Dimension Constraint`, V-04 `The Dimension Rule` (conversational tone), V-05 `Dimension Rule` (with explicit substitution mappings: "Reliability → use Correctness").

**C-14 implementation:** All five put Stage 3.5 as a named stage between Stage 3 and Stage 4 with the exact token pair. V-01 calls it a "binary and irreversible" decision; V-02 makes it a "ruling" issued before handoff; V-03 calls it a "stop gate" in the Gate Sequence Overview table so its halt behavior is visible at the top; V-04 frames it conversationally ("answer one question"); V-05 includes `COMMIT-STAGE-3-FLAGGED` in the Role Architecture diagram so the halt path is declared before any stage runs.

**C-11 (prose blocks):** Stage 4 is prose blocks in all five — "tables are prohibited in this stage" in V-01/V-03, "prose only — no tables" in V-02, template with worked examples in V-04, "prose removes the cell-width pressure" note in V-05.
