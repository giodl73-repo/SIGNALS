### V-02 · Phrasing Register — Epistemic Question Framing

### V-03 · Inertia Framing — Two-Register Question Structure
**Axis / Hypothesis:** C-48 alone. **The C-48 structural requirement:** V-05 R14 had null questions first but mixed with area framing, no register labels, no close markers. V-03 adds: explicit `NULL REGISTER` header, explicit `NULL REGISTER CLOSE` paragraph, explicit `DEPARTURE REGISTER` header, explicit `DEPARTURE REGISTER CLOSE` paragraph. The null baseline must be established as a closed prior section before departure questions open.

### V-04 · Combined: Role Sequence + Phrasing Register
**Axis / Hypothesis:** C-47 + C-46. Phase boundary (V-01 structure) with epistemic questions throughout (V-02 framing). Eight CAUSAL PHASE questions span Area 2 (Q1–Q3) and Area 3 (Q4–Q8); Q4–Q5 naturally address the null state before Q6–Q8 address departure properties — testing whether C-48 emerges without explicit register labels.
