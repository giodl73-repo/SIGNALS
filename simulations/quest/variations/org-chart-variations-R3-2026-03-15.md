**Key design decisions:**

**Single-axis first.** V-01 and V-02 each add one section to the proven V-04/R2 foundation. If either loses one of the six R2 criteria when the new section is appended, that interaction is immediately locatable.

**V-03 tests C-15 ambiguity.** V-04/R2 passed C-15 by embedding mechanism reasons inside the "Case for Staying Flat." Under a strict rubric reading, the coordination inventory must be a *separate* labeled section. V-03 makes it explicit as "How We Coordinate Today" — separating *why flat works* (defense) from *what mechanisms exist* (inventory). If V-03 and V-04/R2 score identically, the separation is not load-bearing. If V-03 is more reliable, it should be adopted into future rounds.

**V-05 tests register.** The imperative DO/DO NOT format has been the backbone since R1 V-04. V-05 converts the identical structural coverage to descriptive prose. If it scores the same as V-04, register is style. If C-14 or C-16 drop (the most semantically precise criteria), the prohibitions in DO/DO NOT are functionally required.
