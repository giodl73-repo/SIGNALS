## R8 Variations — prove-interview

Five complete prompt variations targeting the three new v8 criteria as a coordinated set.

### Variation Axes

| Var | Axis | Hypothesis |
|-----|------|-----------|
| **V-01** | Lifecycle emphasis | Named decision system + binary C-30 gate checks eliminate PARTIAL-prerequisite activation |
| **V-02** | Output format | Schema-contract rule forces C-29 everywhere; decision-system as an instantiated table forces C-28 |
| **V-03** | Phrasing register | Conversational/descriptive framing — tests whether imperative language is load-bearing for C-28/C-29/C-30 |
| **V-04** | Role sequence + Inertia framing | Status-quo competitor named as structural artifact; C-28 INCONCLUSIVE rule includes inertia-anchoring notes |
| **V-05** | Output format + Lifecycle | C-28/C-29/C-30 as a coordinated structural trio: pre-printed taxonomy schema + example rows throughout + 0-D master gate table |

### Key structural differences from R7

**C-28 (new)**: All five variations now declare the moment-label taxonomy as a *three-label decision system with explicit rules for each label* — including a named decision rule for INCONCLUSIVE — before the first transcript. R7 variations applied labels with format templates but never declared the governing system.

**C-29 (new)**: V-01, V-02, V-04, V-05 include populated example rows immediately after every table schema declaration. V-02 and V-05 enforce this as a document-level contract rule. V-03 includes one example row in the evidence table guidance.

**C-30 (new)**: V-01 and V-05 encode a master prerequisite gate table (Section 0-D) that checks each dependent mechanism (C-25 bridge, C-26 vocabulary audit, C-27 dual taxonomy) against its prerequisite with an explicit PARTIAL-stops-here note. V-04 has inline prerequisite checks. V-03's conversational framing has guidance-level (not gate-level) prerequisites — the variable to test.

### Primary discrimination prediction

V-05 should be the strongest structural guarantee on all three new criteria simultaneously. V-03 is the test case: if conversational framing underperforms on C-30 specifically (where gate language is load-bearing), that confirms imperative phrasing is the structural enforcer for prerequisite confirmation.
