### Key structural differences across variations

**C-15 mechanism:**
- V-01/V-03: Phase 0 with explicit steps + `[PHASE 0 COMPLETE]` marker
- V-02/V-04: Numbered checklist with contract table + hard gate prose
- V-05: Compressed 4-field block + gate per artifact

**C-16 mechanism:**
- V-01: Ledger with propagation rule stated; close-time audit confirms
- V-02/V-04: Ledger with fill-in fields; verbatim match enforced in proposal table and pitch
- V-03: Imperative "call it Option 0: [name]" + downstream confirm instruction
- V-05: Propagation rule in Phase 0; options table verbatim; close check
