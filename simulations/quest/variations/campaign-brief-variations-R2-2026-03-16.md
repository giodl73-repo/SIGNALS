## campaign-brief — V-01 through V-05 / Round 2

### Variation axes selected

- **Single-axis:** Session delta (V-01), Commit-risk grounding (V-02), Confidence engineering (V-03)
- **Combined:** R1 winner synthesis (V-04), Full aspirational sweep (V-05)

---

### V-01 — Session delta
**Axis:** DELTA block positioned second (after TOPIC, before STRATEGY).
**Hypothesis:** Making session progress the *frame* for reading the current dashboard — not an afterthought. Requires the block even on first run ("prior_brief: NONE") so C-09 is never silently skipped. Cleanest single-axis C-09 test with V-05 base structure.

---

### V-02 — Commit-risk grounding per gap
**Axis:** Every blocking entry in STATUS requires two named fields: `Assumption:` (what we accept if we commit today) + `Consequence:` (what happens if wrong). A gap that fails the Consequence test is reclassified as optional.
**Hypothesis:** V-04 in R1 applied inertia as a theme. V-02 makes it *item-level and mandatory*. Strongest C-11 mechanism in the set — the reclassification rule forces C-07 quality by structural consequence, not instruction.

---

### V-03 — Confidence dimension rubric
**Axis:** Three-dimension scoring table in STORY: signal breadth (1-3), signal depth (1-3), gap severity (1-3). Overall confidence derived from average with defined thresholds.
**Hypothesis:** C-10 was FAIL in all R1 variations because confidence appeared as assertion or prose hedge. A derived score makes it deterministic and auditable. Trade-off: watch whether the rubric table crowds the STORY block (C-12).

---

### V-04 — V-05 blocks + V-04 inertia (R1 structural winner + R1 content winner)
**Axis:** V-05's terminal block structure + V-04's inertia framing. Adds `inertia_cost:` to VERDICT and `Inertia risk:` per blocking gap in STATUS.
**Hypothesis:** These two axes never co-occurred in R1. Expected score ~100+/130. Remaining ceiling: no DELTA (C-09 FAIL), C-10 PARTIAL at best. This is the "safe combination" — verifies R1 winner synergy before adding aspirational complexity.

---

### V-05 — Full aspirational sweep
**Axis:** V-05 base (terminal blocks) + DELTA block (C-09) + confidence dimension table in STORY (C-10) + per-gap inertia fields in STATUS (C-11). C-12 and C-13 are already structural in V-05 base.
**Hypothesis:** First variation targeting all 13 criteria simultaneously. Ceiling test: can all 13 co-exist without overloading the prompt? Watch for STORY block inflation from the confidence table and STATUS verbosity from inertia fields.
