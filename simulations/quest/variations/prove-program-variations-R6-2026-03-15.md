## prove-program R6 -- V-01 through V-05

**Context**: R5 V-05 confirmed 160/160. R6 rubric is unchanged. All five R6 variations are projected at 160/160 -- the goal is demonstrating ceiling stability across genuinely different structural axes.

---

### Variation axes chosen

| V | Axis | What changes |
|---|------|-------------|
| V-01 | **Role sequence** | 3-role (PLANNER/EXPERIMENTER/SYNTHESIZER) -> 4-role (FRAMER/PLANNER/RUNNER/EVALUATOR). FRAMER owns hypothesis commitment; PLANNER owns experiment design. GATE-1 is a hard role boundary. |
| V-02 | **Output format** | Phase/role labels -> numbered procedure (Step 1-10). Gates named by step position (GATE-2A, GATE-7A). Tests if ceiling is format-agnostic. |
| V-03 | **Phrasing register** | Third-person formal -> second-person imperative + question-answer frame. Every section header is a direct question ("What do you believe is true?"); body is the answer format. |
| V-04 | **Lifecycle emphasis + Inertia framing** | `prove-topic` capability map table as structural spine, consulted at every phase boundary. Explicit "You are crossing Phase N -> Phase N+1" narration at every gate. |
| V-05 | **Combined** | V-01 role sequence + V-03 register + V-04 inertia framing, all over the R5 V-05 atomic pattern floor (C-20 + C-21 + C-22). |

### Key structural invariants preserved across all 5

- **C-20**: Boolean table with `PASS (A AND B = TRUE) / FAIL (A = FALSE OR B = FALSE)` before the experiment plan in every variation
- **C-21**: Dedicated `GATE-CAL` at findings -> synthesis with explicit `FAIL: all findings rated HIGH` (uniform-label) condition
- **C-22**: Citation contract clause embedded locally in each consumer block's INPUT section -- not a global preamble rule -- in every variation

The rubric coverage table at the end maps all 22 criteria for each variation. The structural divergence (role naming, step numbering, grammatical person, inertia prominence) does not touch any criterion.
