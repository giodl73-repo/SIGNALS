## Round 6 Variations -- Summary

| Variation | Axis | New Criterion | Key Structural Addition |
|-----------|------|---------------|------------------------|
| **V-01** | Lifecycle emphasis | C-20 single-axis | **COMMITMENT DECLARATION** block before each role's vector table; declares per-vector structural assumptions before analysis starts; Phase 3 Gate checks that every NOT POSSIBLE verdict traces back to a named prior assumption -- two-point contract enforced |
| **V-02** | Inertia framing | C-22 single-axis | Replaces the monolithic five-vector table with **five per-vector mini-blocks**, each opening with a `> Design Assumption:` statement; verdict must say `NOT POSSIBLE -- Assumption Confirmed: [fact]` or `POSSIBLE -- Assumption Refuted: G-##`; a verdict without a prior assumption in the same block is structurally invalid |
| **V-03** | Phrasing register | C-23 single-axis | Introduces **No-Finding Declaration format** (`Checked / Confirmed / Conclusion`) required at every level: vector NOT POSSIBLE verdicts, per-role clean sharing rule findings, gap-free ENTITY CLOSURE blocks, team no-dependency confirmations; bare negatives fail the format check |
| **V-04** | Output format + Role sequence | C-21 + C-24 combined | Renames the 4th column to **`Structural Basis`** with an explicit rule that it holds only one structural fact sentence (no verdict language, no evidence prose); the ESCALATION ROLL-UP's `Mechanisms cited` field names every vector and **restates each structural fact inline** -- writing "see table above" is a format violation |
| **V-05** | Combined | C-20 + C-21 + C-22 + C-23 + C-24 | Full five-criterion stack: COMMITMENT DECLARATION + inertia mini-blocks with `Assumption Confirmed/Refuted` verdicts + No-Finding declarations at every level + isolated Structural Basis column + self-contained roll-up restating mechanisms inline |

**Discrimination logic under v6:** V-01 through V-04 each target one independent structural gap. V-05 stacks all five. The three R5 variations that tied at 100 (V-03, V-04, V-05 of R5) are now differentiated because: C-20 separates pre-committed assumptions from post-hoc mechanism citations; C-21 separates prose-embedded mechanism from a column-isolated structural fact; C-22 separates free-form NOT POSSIBLE from assumption-confirmed NOT POSSIBLE; C-23 separates asymmetric evidence from symmetric; C-24 separates back-referencing roll-ups from self-contained ones.
