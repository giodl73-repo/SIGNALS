## corps-committee R2 — 5 Variations

### What changed from R1

R1 scored 86–95 across all five variations. Every variation got PARTIAL on C-08 (charter-grounded specificity) for the same root cause: no mechanism forced Phase 3 voices to actually differ by charter orientation. R2 targets the three new v2 aspirational criteria:

| Variation | Axis | New criteria targeted |
|-----------|------|-----------------------|
| **V-01** | Role-distinctive voices | C-09 — `ORIENTATION-LENS:` line opens every voice block, quoting Phase 0 charter phrase verbatim; prose must follow from it; distinctiveness rule validates at least 3 non-transplantable voices |
| **V-02** | Structural-table format | C-10 — STANCE MANIFEST and ACTION ITEMS both become Markdown tables with named columns (`Role\|Stance`, `Owner\|Action\|Deadline`); blank cells fail; Phase 4 counts table rows not prose |
| **V-03** | Injection-default framing | C-11 — INERTIA CONTINUITY BRIDGE recast as a YES-gate; Inertia-Advocate is always present by default; YES requires named role + verbatim orientation quote; explicit "YES without qualifying evidence fails" language + motivational rationale |
| **V-04** | C-09 + C-10 combo | Orientation anchoring and table structure together, without the injection-default change |
| **V-05** | Full integration | All three mechanisms co-present; predicted ~97 if the model produces genuinely orientation-grounded voices |

### Open question for quest-score

C-09 is the hardest to earn. V-01/V-04/V-05 create structural pressure with the `ORIENTATION-LENS:` line and the role-distinctiveness VALIDATE, but whether the model actually generates non-interchangeable voices — rather than generic concerns with an orientation header bolted on — is what the scoring round will reveal.
