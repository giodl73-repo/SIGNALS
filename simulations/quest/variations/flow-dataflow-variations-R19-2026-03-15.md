## Summary

**Target criteria**: C-38 (CMD as scaffold entry zero), C-39 (CMD Binding column on every scaffold row), C-40 (T-07 Running Total column in every PG-NN gate), C-41 (dual-count checkpoint [N] + [M]).

**What R18 left exposed**:
- CMD Binding appeared on high-risk rows only, not all rows (C-39 gap)
- PG-NN gates had GO/NO-GO but no T-07 Running Total column (C-40 gap)
- Checkpoints had [N] only — no [M] (C-41 gap)
- CMD positional primacy was a V-01 behavior, not a named CMD prohibition (C-38 gap)

### Variation axes

| Variation | Axis | Core mechanism |
|-----------|------|----------------|
| **V-01** | Lifecycle emphasis | Three-tier accumulation protocol as a named lifecycle device — Phase 1 declares all three tiers (per-stage, per-phase, post-trace) with dedicated CMD entries. Dual-count and running-total are spine, not format. |
| **V-02** | Output format | Scaffold column explosion: CMD Binding + RT-Col? (YES/NO) + Satisfies Criterion on every row. RT-Col? = YES on every PG-NN entry pre-declares the T-07 Running Total requirement before any gate is produced. |
| **V-03** | Role sequence | Two-role Finance/Operations split. CMD register has a Role Owner column — every CMD entry is attributable to a named role. CMD Binding + Role Owner on every scaffold row. T-07 running total is the handoff continuity contract at PG-02. |
| **V-04** | Phrasing register | Coaching-imperative: teaches the arithmetic explicitly ("M at stage k = M at stage k-1 + N at stage k"). CMD still entry zero; CMD Binding still on every row; but language is instructional rather than command-and-prohibit. |
| **V-05** | Inertia framing | Incumbent baseline before scaffold. Triple-count checkpoint: [N], [M], and IB-NN steps covered. PG-NN gates carry both T-07 Running Total AND IB-NN Coverage columns. C-40/C-41 failure is simultaneously a C-16 traceability gap. |

**Key R19 innovations beyond R18**:
- CMD-18/CMD-19 (V-01) — named prohibitions specifically for `[N]-only` checkpoint and missing running-total gate column
- RT-Col? scaffold column (V-02) — forces T-07 Running Total into PG-NN Mandatory Columns at declaration time
- Role-attributed CMD slices with handoff gate as running-total continuity contract (V-03)
- Explicit accumulation formula as coached arithmetic, not just a format rule (V-04)
- Triple-count checkpoint weaving IB-NN coverage into accumulation (V-05)
