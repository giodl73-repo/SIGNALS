## org:committee Round 3 — V-01 through V-05

Five complete, runnable variations. Here's the design logic:

### Variation Map

| Variation | Primary Axis | Key Hypothesis |
|-----------|-------------|----------------|
| **V-01** | Role sequence (three-tier sort) | R2 V-01 already passed C-11 at 91. Patching its two gaps (C-15 STANCE: label, C-14 tally) while fixing C-08 resolution paths should push past 98. |
| **V-02** | Lifecycle emphasis (investigation phase) | R2 V-02 scored 93 — best in round — but failed C-11 and C-13. Adding tier sort + structural STANCE: to the proven investigation base should achieve all 7 aspirational criteria. |

| **V-04** | Inertia framing (switching-cost as first-speaker) | Instead of a pre-meeting investigation phase, the inertia-advocate speaks first and tags findings (INERTIA-FINDING-A/B). All other voices must acknowledge the tags. Creates sharper challenge-response dynamics than a passive artifact. |
| **V-05** | Combined (investigation + three-tier sort + structural labels + tally) | Four orthogonal enforcement mechanisms targeting four distinct criteria — no interaction effects. First R3 variation targeting all 7 aspirational criteria simultaneously. |

### New v3 Requirements Coverage

Every variation addresses both new criteria:
- **C-14** (vote tally before minutes): All five include a `TALLY: N APPROVE · N CONDITION · N BLOCK` line with explicit placement enforcement
- **C-15** (STANCE: as structural label): All five require `STANCE: [APPROVE / CONDITION / BLOCK]` as a standalone field before prose
- **C-08 fix**: All five now require dissents to include an explicit resolution path with a named authority, not just objection capture

### Predicted Composite Order
V-05 > V-02 > V-01 > V-03 > V-04 (V-04 trades investigation phase depth for speaker-based inertia, which is the riskiest exchange)
