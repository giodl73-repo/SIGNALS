## Round 2 Variation Summary

All five variations target the two new v2 criteria: **C-11** (threshold sourcing provenance) and **C-12** (coverage self-verification).

### Single-axis (3)

**V-01 — Source Citation Column in TABLE A**
Axis: Output format. Adds a `Source` column immediately adjacent to `Limit` in TABLE A. Attribution becomes a form constraint at the moment a number is written. Tests whether adjacent-cell enforcement produces specific section names or generic "PA docs" placeholders.

**V-02 — Mandatory Terminal Self-Verification Section**
Axis: Output format. Closes every output with a pre-printed COVERED/PARTIAL/NOT COVERED checklist against all 12 criteria. Tests whether the model self-reports gaps honestly (earning C-12 partial credit) or backfills missing content to avoid marking checkboxes FAIL.

**V-03 — Source-Register Phase Before Tier Inventory**
Axis: Role sequence. A PHASE 0 commits the model to documentation sources _before_ it writes any numbers. Any TABLE A number not traceable to a Phase 0 entry must be replaced with `unknown [undocumented]`. Tests whether pre-commitment to sources reduces invented thresholds more effectively than V-01's adjacent-column mechanism.

### Combined (2)

**V-04 — Source Column + Self-Verification**
C-11 enforced at TABLE A completion; C-12 enforced at terminal position. The two mechanisms operate at different structural points so neither crowds the other. Preserves all proven R1 table patterns. Tests whether both new criteria can be addressed simultaneously without compressing C-03/C-08.

**V-05 — Source Register + Inertia + Burst/Uniform + Mitigation + Self-Check**
Full integration of all proven patterns with both new criteria. Extends the inertia preamble to name **undocumented thresholds** as the third failure mechanism alongside load-shape blindness and finding-without-fixing. All 12 criteria have structural or motivational enforcement. Predicted composite: ~100/110.

---

**Key design decision:** V-01 and V-03 test the same criterion (C-11) through different enforcement mechanisms — adjacent-column vs. pre-commitment — so the round can determine which approach produces higher source quality before combining them.
