**Design notes:**

- All R15 open questions are closed (Q1: PERMISSIVE). R16 has no open questions to resolve — variations confirm new criteria in unexplored configurations and push toward the new ceiling.
- V-01 is the canonical C-42 reference: cleanest simultaneous satisfaction of all three new criteria in the reference register. Run this first.
- V-02 probes the one register surface not yet tested: RFC-modal MUST/SHALL. Both forms it uses ("every call MUST be documented" / "SHALL become ship-blockers") are unconditional — expected to pass C-40, but the modal surface is a free variable until run.
- V-03 measures whether C-41 has a saturation limit — prior confirmation (R15 V-04) used moderate inertia framing; V-03 makes inertia the dominant narrative.
- V-04 is the first C-40 + C-41 combination — the two axes are structurally orthogonal (register is a surface property; framing is a content property), so no interaction is expected.
- V-05 reuses the R15 V-05 five-row term set (expose-call / silent-entry / shadow-system / delegation-chain / burst-start) and adds C-42 WHY form, making it the first 262-pt ceiling attempt. Stage 3 gap type column updated to include `burst-start-undefined` as a valid gap type for the new obligation.
