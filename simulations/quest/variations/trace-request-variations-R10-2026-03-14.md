**Single axes picked: phrasing register (x2 regression), lifecycle emphasis, output format. V-05 combines lifecycle + output.**

**V-01/V-02** are regression tests. C-26 and C-27 are both new rubric criteria in v9 — if either softened variation still scores full, the DISQUALIFIER wording needs hardening.

**V-03/V-04/V-05** target **ES-3** (the un-promoted C-28 candidate from R9): VT# scope string propagates coherently across Step 3 Auth Check → Step 8a Invoked → Step 11 Parameters. The three variations offer independent structural expressions — prose confirmation (V-03), cell-verifiable table (V-04), both together (V-05). V-05's key property: a mismatch between the inline prose confirmation and the Step 8c Match? = N cell is itself a structural signal that prose alone could not surface.

**C-28 promotion candidate**: requires both C-26 and C-27 confirmed load-bearing in R10 scoring plus consistent ES-1/ES-2 evidence from V-03/V-04/V-05.
