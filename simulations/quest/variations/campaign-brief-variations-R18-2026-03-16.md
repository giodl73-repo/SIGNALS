## Variations -- campaign-brief / Round 18

**R18 investigation:** Does C-37 PASS produce complete-parity companion output under double-elision, or depth-asymmetric output? If asymmetric, does an explicit parity constraint close the depth gap (C-38 candidate)?

**Absent-state rule extension:** `"...the absent [companion]'s mandate is independently operative even when its execution note is not in context."`
**Expected:** 360/370 (C-37 PARTIAL Band 1 -10)

### V-02 — C-37 PARTIAL Band 2 form-A
**Axis:** Conditioned execution instruction — `"determinable from this clause body's context"` scope qualifier.
**Absent-state rule extension:** `"...is independently operative...execute the companion's mandate where its obligations are determinable from this clause body's context."`
**Expected:** 360/370 (C-37 PARTIAL Band 2 -10). Model must assess scope before executing — inference burden reintroduced at scope-determination step.

### V-03 — C-37 PARTIAL Band 2 form-B
**Axis:** Conditioned execution instruction — `"inferrable from this clause body alone"` scope qualifier with explicit annotation mandate.
**Absent-state rule extension:** `"...execute the companion's mandate to the extent its obligations are inferrable from this clause body alone, and note the scope boundary where inference is insufficient."`
**Expected:** 360/370 (C-37 PARTIAL Band 2 -10). Sub-band candidate: Band 2-A (silent truncation at scope boundary) vs. Band 2-B (annotated truncation).

---

### V-04 — C-37 PASS minimum-sufficient (R17 V-05 reprised) — **primary R18 behavioral probe**
**Axis:** Unconditional execution mandate, no parity constraint.
**Absent-state rule extension:** `"...is independently operative even when its execution note is not in context and must be executed as if its execution note were present in this context."`
**Expected:** 370/370 (C-37 PASS ceiling). **R18 probe:** observe companion output depth under double-elision vs. full-context baseline. Full-parity → C-38 not necessary. Depth-asymmetric → V-05 is C-38 candidate.

---

### V-05 — C-37 PASS + explicit parity constraint — **R18 C-38 probe**
**Axis:** Unconditional execution mandate extended with `"at full depth, at parity with what would be produced if the companion's execution note were present in this context."`
**Absent-state rule extension:** `"...must be executed as if its execution note were present in this context, at full depth, at parity with what would be produced if the companion's execution note were present in this context."`
**Expected:** 370/370 (C-37 PASS ceiling; parity constraint above-ceiling for v17 — not rubric-scorable, but observable under double-elision comparison against V-04).

---

**Single axis across V-01–V-03:** absent-state rule activation depth (operability-only → conditioned execution → conditioned execution + annotation).
**V-04 vs V-05 axis:** parity constraint presence — "as if present" (V-04) vs. "as if present, at full depth, at parity" (V-05). If V-04 already achieves full-depth companion output, V-05 is redundant and C-38 is closed. If V-04 produces abbreviated companion output, V-05 is the C-38 structural candidate.
