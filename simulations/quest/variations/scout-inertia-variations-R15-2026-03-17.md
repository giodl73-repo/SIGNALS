## Key R15 Design Decisions

**C-49 canonical form** (used in V-01/V-02/V-03/V-05, tested alone in V-04):

```
HAVE BOTH Q3 (FM-ACTOR-BRIDGE) AND Q4 (FM-TRIGGER-BRIDGE) [COMPOUND CLASS NOUN] BRIDGES BEEN POPULATED?
```

The stacked form adds the trailing compound class noun after both parenthetical annotations. This satisfies C-49 (per-artifact type), C-46 (compound threading in class noun), C-45 (artifact IDs named), and C-43 (BOTH = count 2) simultaneously.

**C-51 two forms tested**:
- Joint: `Return to Stage 2A if Q3 shows N; return to Stage 2B if Q4 shows N.` (V-01/V-02/V-04)
- Per-artifact: `IF Q3 SHOWS N, RETURN TO Q3 (SECTION 1A). IF Q4 SHOWS N, RETURN TO Q4 (SECTION 1B).` (V-03/V-05)

**C-52 two forms tested**:
- Number-only: `BASED ON EVIDENCE IN SECTIONS 1-3` (V-01/V-02/V-03/V-05)
- Number + heading title: `BASED ON EVIDENCE IN STAGE 3 (WORKAROUND PROFILE) AND STAGE 4 (SWITCHING COST ANALYSIS)` (V-04)

**Open structural question**: V-04 uses `Q3 (FM-ACTOR-BRIDGE) AND Q4 (FM-TRIGGER-BRIDGE)` without a trailing class noun. C-49 passes (parentheticals present). C-46 may fail at the gate interrogative if the compound threading vocabulary must appear there. V-01/V-02/V-03/V-05 avoid this risk by stacking parentheticals with the compound class noun.

**Predicted scores**: All five 44/44. V-04 carries a C-46 risk at the gate interrogative if the trailing class noun is required there.
