## Structural innovations in this round

**C-21 exemption clarity**: V-01 and V-02 both make the exemption case structurally visible — when all three receipts are VERBATIM on first pass, CHART-WRITTEN follows directly with no TRANSCRIPTION-CLEAR required. V-02 names this as PATH-A vs PATH-B, making the two output shapes distinguishable.

**Phase reordering in V-04**: PROFILE precedes CONTRACT (rather than following CONTRACT-SEAL as in all prior rounds). Resistance profiles become the source from which contract artifacts are assembled, not an independent phase. This elevates C-15/C-16 from quality criteria to structural anchors.

**V-03 produces aspirational evidence**: The deferred R5 axis is now fully developed. Each criterion from C-09 through C-21 is described as a shape constraint on output, not a behavioral instruction. The Shape A / Shape B pattern for TRANSCRIPTION gives the model a visual reference for what correct output looks like in both the all-VERBATIM and post-rewrite cases.
