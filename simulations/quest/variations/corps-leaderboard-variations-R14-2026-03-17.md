### Variation summary

**V-01** — Single axis: phrasing register. Health blocks shift from 5-row achievement tables to field-report prose paragraphs with bolded achievement names inline. Seed O adds a confidence band after the Team Insight. 3-role pipeline.

**V-02** — Single axis: output format. Achievement tables gain a `Progress` column with 10-character ASCII bars (`[######----] 2/3`). Nearest-miss table gains a `progress` column with the same bars. Seed M adds `[closes: ...]` gap-binding on every action row. 3-role pipeline.

**V-03** — Single axis: role sequence. Leaderboard computation moves from Publisher (final role) to Analyst (after Health Phase), making ranked contributor data with a velocity column (Seed R) available to the Strategist at action-writing time. Publisher role reduced to collaboration signal + reconciliation + insight only. 4-role pipeline.

**V-04** — Combination: inertia-first lifecycle + tiered actions (K) + Seeds M+N. New element: milestone proximity ladders appear inline after every NOT YET milestone row, breaking the remaining precondition into individual action counts. Tiered actions carry `[closes: ...]` annotations. 4-role pipeline.

**V-05** — Full integration: 5-role (Archivist → Assessor → Inspector → Strategist → Narrator) + all R13 seeds (A+B+H+I+K+L) + all R14 seeds (M+N+O+R+S). Inspector adds echo detection (Seed S) checking `healthy-momentum` + `Solo Pioneer` mutual exclusions. Narrator leaderboard has velocity column (R), reconciliation uses non-retracted/non-echo-corrected flags, insight carries confidence band (O) + projection test (L).

**All 5 expected: 23/23 → 100.00 against v11.**

The v12 ceiling-breaking opportunity is a variation that adds a pattern across two or more of Seeds M, N, O, R, S that existing criteria do not capture — most promising candidates: Seed N (milestone proximity ladder, not covered by C-21/C-23 which only address achievement-level proximity) and Seed S (echo detection, orthogonal to both C-27 and C-30).
