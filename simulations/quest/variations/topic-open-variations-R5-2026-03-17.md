| Variation | Axis | C-21 | C-22 | C-23 | C-24 |
|-----------|------|------|------|------|------|
| **V-01** | Phase-pervasive tool citation | Deep (7 sites) | Min (vocab block only) | Min (Phase 2 only) | Present |

**Design decisions:**

- Every variation passes C-21 at minimum via the Phase 1 collision-check tool citation
- Every variation passes C-22 at minimum via comparative framing on the vocabulary block's reading-order instruction
- Every variation passes C-23 at minimum via a labeled decision table before Phase 2
- Every variation passes C-24 via a 4-column CONSTRAINT SURFACE table (or the 5-column DEFAULTS TABLE in V-05)

**Distinguishing excellence targets:**
- V-01 will show the highest C-21 depth score — tool citations on every structural element including Phase 2 exit gate items and Phase 5 checklist items
- V-02 will show the highest C-22 depth — comparative framing on all four ordering instructions (vocab block, ROW TEMPLATE copy, rationale order, self-check timing) each contrasting both paths
- V-03 will show the highest C-23 depth — all five phases have named decision tables (Phase 1 collision, Phase 2 order, Phase 3 gate structure, Phase 4 output write, Phase 5 check timing)
- V-04 tests whether combining C-21+C-22 at every instruction creates coherent density or cognitive overload
- V-05 tests whether DEFAULTS TABLE compactness (C-24) and per-phase decision tables (C-23) are complementary or redundant when both present at full depth
