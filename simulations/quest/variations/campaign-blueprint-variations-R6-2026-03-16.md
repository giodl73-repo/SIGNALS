inertia-grounded labels ("Factual — names inertia problem"). Each Must-have carries an
inline friction tag. Per-site reminders use compact labeled lines. Tests whether 158 is
achievable at skeleton density. **Predicted: 158 if compact forms hold; 148 if C-23 or
C-24 fall to PARTIAL.**

### V-05 — Full Convergence *(combination)*

Named INERTIA MODEL entity (V-03) + SCOUT TRACEABILITY TABLE (V-02) + three-field
per-site reminders referencing INERTIA MODEL fields by name. Maximum structural
explicitness. Tests whether combining all three mechanisms simultaneously creates
instruction friction or degraded C-09 quality while maintaining 158. **Predicted: 158.**
