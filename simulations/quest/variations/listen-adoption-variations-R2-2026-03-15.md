## Round 2 Variations — listen-adoption

**Rubric target:** v2 (C-01 through C-13)

### Variation Axes

| Axis | Target criteria | Hypothesis |
|------|----------------|------------|
| Champion displacement annotation | C-12 | Without a required column, champion tables describe bridging but skip the displacement question — what argument does this champion counter? Making it a fill-contract column forces every champion from passive roster to active displacement plan. |
| Three-axis gap audit block | C-13 | Outputs satisfy two of three gap dimensions silently. An independent audit block after Phase 5 certifies all three axes — role completeness, temporal completeness, causal specificity — separately, so satisfying two cannot substitute for the third. |
| Preamble propagation with citation markers | C-11 (stronger than R1 V-03) | R1 V-03's "must reference" instruction allows soft compliance — incumbent named once in passing. Explicit `[CITE: THE INCUMBENT]` markers per downstream section make omission structurally visible as an unfilled slot rather than a missing sentence. |

### Single-axis → Combined → Full

| Variation | Axes | Primary target |
|-----------|------|---------------|
| **V-01** | Champion displacement annotation only | C-12 |
| **V-02** | Three-axis gap audit block only | C-13 |
| **V-03** | Preamble propagation with citation markers | C-11 |
| **V-04** | V-01 + V-03 | C-12, C-11 combined: named incumbent gives champions a concrete argument to counter, making the column easier and more specific |
| **V-05** | V-01 + V-02 + V-03 | C-11 + C-12 + C-13: preamble provides the named thread, audit certifies it was maintained across all three dimensions, champion column makes Axis C of the audit passable |

### Key structural innovations vs R1

- **V-01**: Adds `Incumbent argument countered` column to champion table — fills the exact gap C-12 describes
- **V-02**: THREE-AXIS GAP AUDIT block is a stand-alone fill contract that audits all 16 roles by name (Axis A), all events by month (Axis B), and all gaps by named cause (Axis C) — making each axis independently falsifiable
- **V-03**: `[CITE: THE INCUMBENT]` markers in SECTION 3, 4, 5, and 6 replace soft "must reference" with unfillable slots — propagation failure is visible
- **V-05**: The three axes are non-overlapping: V-03 provides the thread, V-02 audits the thread was propagated across all dimensions, V-01 makes Axis C of V-02's audit passable with named displacement claims
