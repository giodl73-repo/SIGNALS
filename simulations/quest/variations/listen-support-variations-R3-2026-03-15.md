## Round 3 Variations — Summary

**Baseline:** R2 V-05 (composite 96.0 — passes C-01 through C-13)
**Gap:** C-14 (phase-anchored body voice) + C-15 (pre-generation constraint declaration with numeric verification)

### Single-axis variations

| Variation | Axis | New angle | Target |
|-----------|------|-----------|--------|
| **V-01** | Lifecycle emphasis | Phase vocabulary pools with per-ticket phrase commitment table recorded before body writing; body must begin with committed phrase | C-14 |
| **V-02** | Output format | CONSTRAINT MANIFEST table (pre-ticket) + VERIFICATION MANIFEST table (post-ticket) with Required / Actual / Status columns | C-15 |
| **V-03** | Role sequence | Role-phase vocabulary matrix: each cell declares exact register for that role × phase; SRE-first generation order | C-14 + C-12 |

### Combined variations

**V-04** (lifecycle + output format): V-01 commitment table + V-02 manifest tables — both C-14 and C-15 satisfied by construction and by form, independently.

**V-05** (full synthesis): Full R2 V-05 structure + vocabulary commitment table in STEP 2B + CONSTRAINT MANIFEST replacing scattered STEP 1 declarations + VERIFICATION MANIFEST replacing STEP 9 Pass 2 lines. Expected composite **100** — the committed phrase is the C-14 body text evidence; the manifest pair is unambiguously the C-15 pre/post structure.

### Design rationale

The R2 V-05 STEP 2B already had role-phase anchor templates. The failure mode C-14 is trying to catch is not generic bodies — R2 bodies were non-generic — but bodies that carry the right severity label without the right phase-register vocabulary *in the text*. The vocabulary pre-commitment step (V-01) closes this by requiring the model to write down the committed phrase before generating the body: the phrase is both the generation constraint and the auditable evidence. The manifest table format (V-02) closes C-15 by making constraint declaration and numeric verification structural landmarks in the output — impossible to skip, impossible to satisfy silently.
