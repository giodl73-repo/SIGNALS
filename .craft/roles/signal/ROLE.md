# Signal Org — Role Registry

Signal's org emerges from its golden prompts. Every stock role named in a skill description is a Signal employee.

## Areas

- **evidence** — the prove team: analysts, researchers, hypothesis builders
- **discovery** — the scout+draft team: PMs, architects, strategists  
- **quality** — the review+trace team: reviewers, testers, tracers
- **adoption** — the listen+program+topic team: support, SRE, PM
- **platform** — the quest+org team: rubric builders, org designers
